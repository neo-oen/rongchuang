# 组件 · 可复用 SketchUp 组件

存放项目中反复使用的设备、构件 `.skp` 组件，保证全屋模型尺寸一致。

## 建议收录的组件

| 类别 | 示例 | 关联文档 |
|------|------|----------|
| 暖通 | 立式风机盘管、新风口、回风口 | 07、08 |
| 热水 | 空气能主机、生活热水箱、缓冲水箱 | 09、10 |
| 门窗 | 系统窗、门洞 | 05 |
| 检修 | 检修口、阀门位置示意 | 18 |
| 构造 | 轻钢龙骨断面、插座盒、穿墙套管 | 02、06、12 |

## 命名

`类别_名称_尺寸或型号.skp`

示例：`暖通_风机盘管_立式_待选型.skp`

## 注意

- 组件内标注「待选型」时，必须在 16 号文档对应条目注明
- 设备最终型号确认后，更新组件并升版本
