# 物料 · 材质与参数

SketchUp 建模用的贴图、材质库，以及与工程手册联动的物料清单。

## 子目录

| 目录 | 用途 |
|------|------|
| `贴图/` | 墙面、地面、设备外壳等 JPG/PNG 贴图 |
| `材质库/` | SketchUp 材质包（`.skm`）或材质定义说明 |
| `清单/` | 物料参数表，与 [16-材料品牌型号库](../融创大观/16-材料品牌型号库.md) 双向同步 |

## 同步原则

- **手册 → 3D**：16 号文档确定规格后，在此建立对应材质/组件参数
- **3D → 手册**：建模过程中发现的尺寸冲突，回写 16 号文档并走 20 号决策流程
