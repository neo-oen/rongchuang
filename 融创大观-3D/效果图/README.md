# 效果图 · 渲染与导出

存放 SketchUp 渲染图、Layout 导出、场景截图，用于方案沟通与留档。

## 子目录

| 目录 | 用途 |
|------|------|
| `整体/` | 全屋鸟瞰、主要视角 |
| `分房间/` | 各房间效果图 |
| `节点/` | 剖面、大样、构造细节 |

## 命名

`YYYYMMDD_场景_视角_说明.png`（或 `.jpg`）
