## 这是什么

本目录与 [`融创大观/`](../融创大观/) 工程手册**并列**，专门存放：

- SketchUp 三维模型源文件
- 效果图与场景导出
- 物料材质、贴图与参数清单
- 可复用设备 / 构件组件

工程手册负责**文字标准、施工规范、验收规则**；本目录负责**可视化表达与物料数字化**，两者互相引用、同步更新。

## 命名规范

### 模型文件（.skp）

```text
YYYYMMDD_场景_版本.skp

示例：
20260627_全屋_v0.1.skp
20260627_主卧_墙体剖面_v0.2.skp
20260627_厨房后平台_设备布置_v0.1.skp
```

### 效果图

```text
YYYYMMDD_场景_视角_说明.png

示例：
20260627_客厅_南视_立式风机盘管位置.png
20260627_外墙_剖面_标准做法.png
```

### 物料清单

```text
物料清单_系统名.md  或  物料清单_系统名.csv

示例：
物料清单_墙体系统.md
物料清单_风机盘管.csv
```

## 工作流程

```text
1. 工程手册确定原则/尺寸倾向
      ↓
2. 在 SketchUp 中建模或更新对应场景
      ↓
3. 物料参数写入 物料/清单/，同步更新 16 号文档
      ↓
4. 节点模型归档至 模型/节点大样/，同步更新 17 号文档索引
      ↓
5. 导出效果图至 效果图/，用于群里方案确认
      ↓
6. 重大变更：工程手册 20 号文档留决策记录，3D 模型升版本号
```

## 版本管理原则

- **不覆盖旧版本**：重大修改另存新版本号（v0.1 → v0.2）
- **模型与手册一致**：手册变更后，对应 3D 场景必须在 7 天内同步或标注「待更新」
- **大文件**：`.skp`、高清渲染图可不进 Git，用 `.gitignore` 排除后本地/网盘备份；目录内保留 README 说明存放位置
- **小文件优先入库**：物料清单 `.md`/`.csv`、组件说明、低分辨率预览图可提交 Git
