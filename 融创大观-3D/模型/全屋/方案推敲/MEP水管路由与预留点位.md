## 1. 文档说明

本文补全 **设备位置之外** 的内容：

- 各类水管 **走向**（干管、支管、立管）
- **管材 / 标高 / 管径** 倾向
- 各区域 **预留了什么出水口**（RD 立管点 + 设备直联）

**权威数据源**：`rebuild_mep_pipes_only.rb`（pipes_v5）+ 本文 JSON。SketchUp 内 PIPES 分组为 **方案示意**，施工前须现场复核。

## 4. 生活水路由

### 4.1 软水冷水（R-SOFT-01）

```text
市政进水 → 水柜 cabinet（前置+中净+中软）
              ↓ out_soft 顶口
         南排 y=48 干管
    ┌────┬────┬────┬────┬────→ RO 进水
    ↓    ↓    ↓    ↓    ↓
 RD-厨房 RD-客卫 RD-主卫 RD-洗衣区  ro
 (122,74)(222,74)(447,74) (29,326)
```

| 项目 | 内容 |
|------|------|
| 水源 | 水柜 `cabinet` 顶部软水出口 |
| 干管 | `cabinet.cx` → 南排 **y=48** → 向东/西分支 |
| 立管 | 至各 RD 坐标 `(x, y)`，顶标 **z_riser**（见 §7 表） |
| RO 分支 | 干管 → `ro.cx` → 立管至厨下 RO 主机 |

### 4.2 生活热水（R-HOT-01）

```text
[B] 生活热水 dhw 南口 out_hot
         ↓
    南排 y=50 干管（比软水高 2SU）
    ├→ RD-厨房 (122,74)
    ├→ RD-客卫 (222,74)
    ├→ RD-主卫：经 dhw_boost 小厨宝串联 → (447,74)
    └→ RD-洗衣区 (29,326)
```

| 项目 | 内容 |
|------|------|
| 热源 | `[B]` 生活热水空气能（与生活冷热水独立） |
| **无全屋热水回水** | 不做小循环（已确认） |
| 主卫特殊 | 热水干管 → **dhw_boost 小厨宝**（南进北出）→ RD-主卫，缩短主卫热水路径 |

### 4.3 RO 纯水（R-RO-01）

```text
软水 → RO 主机 ro（厨下）
              ↓ 纯水 out
         横管 z=97, y≈82
              ↓
         管线机 pipeline 西口
```

| 项目 | 内容 |
|------|------|
| 进水 | 软水至 RO（经南排分支） |
| 出水 | 纯水专用 PE 管至 **管线机** |
| 不经 RD | 厨房饮用水独立链路 |

### 4.4 废水排水（RO 浓水 + 软水再生）

**去向：厨房下水汇总 DRAIN-HUB-KIT**（水槽下方，与 RD-厨房同区）

```text
RO 浓水口 ── DN20 PVC ──┐
软水再生口 ── DN25 PVC ──┼→ DRAIN-HUB-KIT → 水槽排水 DN50 → 污水立管
厨房水槽下水 ────────────┘
```

| ID | 来源 | 管径 | 说明 |
|----|------|------|------|
| R-DRAIN-01 | RO 浓水 | DN20 | 制水时持续小流量 |
| R-DRAIN-02 | 软水再生 | DN25 | 再生时间歇大流量 |

**禁止**接入雨水、洗衣机地漏或不明管道。

### 4.5 软水→[B]（R-SOFT-02）

```text
南排 y=48 → V03 → [B] dhw 南侧冷水进（DN20）
```

### 4.6 市政入户（R-MUN-01）

```text
管井/玄关 → V01（DN25 总阀）→ 前置 → cabinet 进水
```

管井位置 **待现场复核**。

### 4.7 厨房/洗衣分支

**R-KIT-01**：RD-厨房 → 水槽冷热水 + 洗碗机软水进  
**R-LAU-01**：RD-洗衣区 → 短分支（≤2 m）→ 洗衣机 + 扫地机

## 5. 两联供二次侧路由

与生活热水 **完全分离**。

### 5.1 空调水 → 5 台 FP（R-AC-01 / R-AC-02）

```text
[A] 两联供 ac_hp 北口 sec_supply
         ↓
    上排 y=148 供水干管
    ├─立管→ fp_living（西墙）
    ├─立管→ fp_bed2（西墙）
    ├─立管→ fp_master（东墙）
    ├─立管→ fp_study（东墙）
    └─立管→ fp_kitchen（北侧）
         ↓ 各 FP 回水
    上排 y=144 回水干管 → [A] sec_return
```

**分支规则**（pipes_v5）：

- FP 中心 **x < 250 SU** → 走 **西墙** `x=x0`
- FP 中心 **x ≥ 250 SU** → 走 **东墙** `x=x1`

| FP | 分支侧 | 连接点 |
|----|--------|--------|
| 客厅 | 西 | fp_living 西墙 |
| 次卧 | 西 | fp_bed2 西墙 |
| 主卧 | 东 | fp_master 东墙 |
| 书房 | 东 | fp_study 东墙 |
| 厨房 | 北 | fp_kitchen 北墙 |

### 5.2 二次侧 → 分集水器（R-AC-03）

```text
[A] 东侧 → fh_manifold 西侧总口（并联，不经过 FP 串联）
```

### 5.3 地暖 FH1–FH5（R-FH-01）

干管以 [`floor_heating.json`](./floor_heating.json) 为准（比 pipes_v5 示意更细）：

| 回路 | 干管 `trunk_via` (SU) | 入口 |
|------|----------------------|------|
| FH1 客厅 | (277,42)→(277,157)→(106,157) | 客厅_南入口；**不经客卫** |
| FH2 次卧 | (277,42)→(296,42) | 次卧_南入口 |
| FH3 书房 | (277,42)→(330,42) | 书房_南入口 |
| FH4 主卧主卫 | (277,42)→(430,42) | 主卧_南入口 → 主卫 |
| FH5 厨房客卫 | (277,42)→(122,42) | 厨房_南入口 → 客卫 |

分集水器 → 干管 → 各房间 **回字形盘管**（见 [03-地面系统](../../../融创大观/03-地面系统.md)）。

## 8. 管材管径倾向

| 系统 | 材料 | 管径 | 保温 | 备注 |
|------|------|------|------|------|
| 软水冷水 | PPR / 不锈钢 | DN20–25 | 可选 | 全屋软水 |
| 生活热水 | PPR / 不锈钢 | DN20–25 | **必须** ≥20mm | 无回水 |
| RO 纯水 | PE 食品级 | Ø6–10 | 无 | 专用，不与冷热水混接 |
| 二次侧 | PPR / 不锈钢 | DN25–32 | **必须** ≥20mm | FP+地暖共用源 |
| 地暖 | PE-RT 阻氧 | Ø16 | 地面绝热层 | 5 路回字 |
| 冷凝水 | PVC | DN25–32 | 无 | 坡度 ≥1% |
