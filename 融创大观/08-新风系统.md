# 08 · 新风系统

> 最后更新：2026-06-28  
> 文档状态：✅ 方案+通用深化（**1 主机 + 5 路分区 + 一体温控联动**）

---

## 0. 系统组成（和 FP 一样 5 路）

本项目 **不是 5 台新风机**，而是：

| 层级 | 数量 | 说明 |
|------|------|------|
| **全热交换主机** | **1 台** | 设备带 `erv`，负责全屋换气与热回收 |
| **新风分区** | **5 路 FA1–FA5** | 与 FP（AC1–AC5）、地暖（FH1–FH5）、温控（TC-Z1–Z5）**一一对应** |
| **电动风阀** | **5 个 MD-Z1–Z5** | 每路分支 1 个，装于分风箱或支管上 |
| **送风口** | **5 处** | 各分区 **低位送风** |
| **排风口** | **5 处** | 各分区 **高位排风**（卫间随主区） |

```text
                    北 ↑
┌──────────────────────────────────────────────────┐
│  erv 全热交换主机（设备带，与 [A][B] 并列）         │
│       ↓ 送风主管                                    │
│  ┌─分风箱────────────────────────────────────┐  │
│  │ MD-Z1 → FA1 客厅    MD-Z4 → FA4 主卧+主卫   │  │
│  │ MD-Z2 → FA2 次卧    MD-Z5 → FA5 厨房+客卫   │  │
│  │ MD-Z3 → FA3 书房                            │  │
│  └─────────────────────────────────────────────┘  │
│       ↑ 回风主管 ← 各区分流回风                    │
└──────────────────────────────────────────────────┘

TC-Z1 ──→ MD-Z1 + AC1 + FH1
TC-Z2 ──→ MD-Z2 + AC2 + FH2   （同一套一体温控，非独立新风面板）
  …
```

**已确认（2026-06-28）**：新风 **电动风阀与调度并入 TC-Z1～Z5**，与 FP、地暖执行器同系统；**禁止**另设 5 块独立新风墙控（除非 TC 套餐自带新风页）。

---

## 1. 目标

- **全热交换**集中新风，**5 区分风**，低位送 / 高位排
- 每区 **电动风阀** 由 **TC-Z1～Z5** 联动
- 与 FP、地暖、吊顶、外墙开孔、冷凝水统一协调

---

## 2. 适用范围

| 包含 | 不包含 |
|------|--------|
| 1×ERV + 5 路 FA + 5×电动风阀 | FP 空调水（07 号） |
| 送/排风口、风管路由倾向 | 厨房油烟机排风（强排，独立） |
| TC 联动逻辑 | 燃气报警联动（待弱电深化） |
| 主机冷凝水 | 卫生间单独排气扇（可选补充） |

**覆盖房间**：客厅、次卧、主卧、书房、厨房 + 玄关随 Z1 溢流；主卫随 Z4、客卫随 Z5。

**暂不单独分区**：阳台、洗衣区、衣帽间 — 通过相邻区溢流补风；若 CO₂ 偏高可后期加排风口。

## 3. 五路分区（与 FP / TC 对照）

> 机器可读：[`fresh_air.json`](../融创大观-3D/模型/全屋/方案推敲/fresh_air.json) · [`hvac_control.json`](../融创大观-3D/模型/全屋/方案推敲/hvac_control.json)

| 分区 | 新风路 | TC | FP | 地暖 | 服务区域 | 设计风量 m³/h |
|------|--------|-----|-----|------|----------|---------------|
| Z1 | **FA1** | TC-Z1 | AC1 客厅 | FH1 | 客厅 + **玄关**溢流 | **90** |
| Z2 | **FA2** | TC-Z2 | AC2 次卧 | FH2 | 次卧 | **55** |
| Z3 | **FA3** | TC-Z3 | AC3 书房 | FH3 | 书房 | **50** |
| Z4 | **FA4** | TC-Z4 | AC4 主卧 | FH4 | 主卧 + **主卫**排风 | **70** |
| Z5 | **FA5** | TC-Z5 | AC5 厨房 | FH5 | 厨房 + **客卫**排风 | **65** |
| **合计** | — | — | — | — | — | **330** |

**主机选型**：全热交换新风机 **350 m³/h** 级（名义风量，留管道衰减裕量；依据 GB/T 18883 人均 ≥30 m³/h、JGJ/T 440 换气次数法，≈120 m² 住宅常用 **250–350 m³/h**）。

## 5. 风口布置（低位送 / 高位排）

### 5.1 原则

| 项目 | 要求 |
|------|------|
| 送风 | **低位**：完成面以上 **300 mm** 或踢脚线上部墙送/地送（避开 FP 直吹通道） |
| 排风 | **高位**：吊顶内或距顶 **200 mm** 内墙排 |
| 气流 | 送 → 人活动区 → 排，形成 **单向流通** |
| 卫间 | **不设独立 TC**；**主卫排风口在 Z4**，**客卫在 Z5** |
| 厨房 | 送风避开灶台正上方；排风可加强（FA5 风量已略高） |

### 5.2 各区分风口倾向

| 路 | 送风 SA | 排风 EA | 备注 |
|----|---------|---------|------|
| FA1 | 客厅 **西墙或北墙** 低位 | 客厅 **南窗上沿吊顶** 高位 | 玄关靠 FA1 溢流 |
| FA2 | 次卧 **东墙** 低位 | 次卧 **北墙吊顶** 高位 | TC 在东墙，风口错开 |
| FA3 | 书房 **东墙** 低位 | 书房 **北墙吊顶** 高位 | 避开书桌直吹 |
| FA4 | 主卧 **东墙** 低位 | 主卧吊顶 + **主卫吊顶** 各 1 | 主卫排风必设 |
| FA5 | 厨房 **西墙** 低位 | 厨房吊顶 + **客卫吊顶** 各 1 | 远离燃气灶 |

**风口规格（通用）**：送风 **Ø160 mm** 可调格栅；排风 **Ø160 mm**；主卫/客卫可加 **Ø100 mm** 辅助排。

## 6. 风管路由

> 详 [`fresh_air.json`](../融创大观-3D/模型/全屋/方案推敲/fresh_air.json) `routes`

```text
erv 主机（设备带）
  ├─ 送风主管 Φ250 → 分风箱（含 MD-Z1～Z5）
  │     ├─ R-FA-01 → FA1 客厅
  │     ├─ R-FA-02 → FA2 次卧
  │     ├─ R-FA-03 → FA3 书房
  │     ├─ R-FA-04 → FA4 主卧（→ 主卫支管）
  │     └─ R-FA-05 → FA5 厨房（→ 客卫支管）
  └─ 回风主管 Φ250 ← 各 EA 汇总
```

| 路由 ID | 分区 | 管径倾向 | 路径摘要 |
|---------|------|----------|----------|
| R-FA-01 | FA1 客厅 | 送 Φ160 / 回 Φ160 | 设备带 → 北过道吊顶 → 客厅 |
| R-FA-02 | FA2 次卧 | 同左 | → 次卧 |
| R-FA-03 | FA3 书房 | 同左 | → 书房 |
| R-FA-04 | FA4 主卧 | 送 Φ160 + 卫 Φ100 | → 主卧 + 主卫支管 |
| R-FA-05 | FA5 厨房 | 同左 | → 厨房 + 客卫支管 |
| R-FA-M01 | 送风主管 | **Φ250** PE 或镀锌 | erv → 分风箱 |
| R-FA-M02 | 回风主管 | **Φ250** | 各 EA → erv |

**施工要点**：

- 风管 **保温**（北向段、穿越非空调区）：橡塑 **≥ 20 mm**，防结露
- 过梁：**过梁器**或局部降吊顶，禁止暴力打梁
- 与 **空调水/冷凝水/强电** 平行间距 ≥ 100 mm，交叉垂直
- 分风箱位置：设备带或 **上排过道吊顶** 检修口可及

## 7. 电动风阀 MD-Z1～Z5

| ID | 分区 | 安装位置 | 规格 | 控制 |
|----|------|----------|------|------|
| MD-Z1 | FA1 | 分风箱出口或 R-FA-01 根部 | **220 V 或 24 V** 常闭型 | TC-Z1 |
| MD-Z2 | FA2 | 同上 | 同左 | TC-Z2 |
| MD-Z3 | FA3 | 同上 | 同左 | TC-Z3 |
| MD-Z4 | FA4 | 同上 | 同左 | TC-Z4 |
| MD-Z5 | FA5 | 同上 | 同左 | TC-Z5 |

**逻辑（由一体 TC 实现）**：

| 模式 | 电动风阀 | ERV 主机 |
|------|----------|----------|
| **通风开** | 本区 **开** | 有任一路开 → 主机 **低/中档**；全关 → 主机 **待机** |
| **通风关** | 本区 **关** | — |
| **制冷/制热** | 可设 **最低通风开度**（防 CO₂ 堆积，如 30% 开度或定时） | 联动低档 |
| **离家** | 全关 | 待机 |

**禁止**：风阀失效常开导致未控区过风；厨房爆炒时关 FA5 送风而不排油烟（油烟机独立）。

## 8. 一体温控联动（TC + FP + 地暖 + 新风）

> 与 [07-风机盘管系统 §7](./07-风机盘管系统.md) 同一套 TC；配置 **`hvac_control.json` v3**

| 分区 | TC | 联动对象 |
|------|-----|----------|
| Z1 | TC-Z1 | AC1 + FH1 执行器 + **MD-Z1** |
| Z2 | TC-Z2 | AC2 + FH2 + **MD-Z2** |
| Z3 | TC-Z3 | AC3 + FH3 + **MD-Z3** |
| Z4 | TC-Z4 | AC4 + FH4 + **MD-Z4**（含主卫） |
| Z5 | TC-Z5 | AC5 + FH5 + **MD-Z5**（含客卫） |

**采购要求**：TC、FP 控制器、地暖执行器、**电动风阀** 必须 **同品牌兼容套餐**（支持 FP+地暖+新风 **三合一** 或总线模块扩展）。

**布线**：

- TC → MD：**≥ 2 芯** 风阀控制线（或总线 2 芯），与 FP 控制线 **分管**
- 设备带 **新风集控模块/端子排** ↔ 各 MD + erv 调速
- 弱电星型，与地暖执行器共用设备带端子排柜

**扩展运行模式**（写入 `hvac_control.json`）：

| 模式 ID | FP | 地暖 | 新风 MD |
|---------|-----|------|---------|
| heat_comfort | 低/关 | 开 | 开（低档） |
| heat_fast | 中 | 开 | 开 |
| cool | 自动 | **强制关** | 开（建议最低档） |
| vent_only | 关 | 关 | 开 |
| away | 关 | 关 | 关 |

## 12. 施工顺序

```text
1. 设备带 erv 基础 + 强电 L-ERV + 室外进排风预留孔（物业批后）
2. 吊顶内 R-FA-M01/M02 主管 + 分风箱 + MD-Z1～Z5
3. 各区分支 R-FA-01～05 + 保温 + 拍照
4. 送/排风口安装（低位/高位）

7. 风量平衡调试 + CO₂/噪声记录
8. 封吊顶前验收拍照
```

## 14. 禁止事项

- 5 台独立新风机（本项目 **1 主机 + 5 路**）
- 新风与 **油烟机、燃气排烟** 混用同一管道
- 送排 **同侧同高**（短路）
- 无电动风阀直吹未使用区
- TC 与 FP **同墙**（07 号已禁）
- 破坏外墙 **隔汽层** 未补强（02 号）
- 未做风量平衡即封吊顶
