## 1. 目标

- 明确外窗 **型式与性能倾向**（已选 **断桥铝 · 经济实用款 · 上下分格**）

## 3. 外窗选型 · 已确认型式（2026-06-28）

### 3.1 意向款：断桥铝 · 经济实用款 · 上下分格

> 甲方意向参考：**上下分格、下固上开** 的断桥铝系统窗（俗称「经济实用款」；下段大固定玻璃，上段开启扇，适合高层观景且 **降低恐高感**）。

| 项目 | 要求 |
|------|------|
| 型材 | **断桥铝合金** 系统窗；**65 系列及以上**；等温线设计倾向 |
| 分格 | **水平横梃** 将窗分为 **下固定 + 上开启** |
| **下段** | **固定玻璃**，尽量大面通透；玻璃 **落地至窗台** 或距完成面 **≤ 100 mm** |
| **上段** | **1 樘/单元内 1–2 扇开启**（外开或内开倒，现场定）；其余固定 |
| 横梃高度 | 完成面以上 **900–1100 mm**（宜与窗台/视线协调；**现场复尺后锁定**） |
| 色 | 室外/室内 **深灰或黑色**（与物业外立面统一为准） |
| 五金 | 多点锁闭；**儿童安全锁**（次卧倾向强制） |
| 纱窗 | **可拆卸金刚网** 或隐形折叠网（逐樘配置） |

**禁止**：下段做可开启扇且无护栏（高层安全）；非断桥普通铝合金；单层玻璃。

> 决策记录：[20-问题决策记录](./20-问题决策记录.md) **2026-06-28-012**

### 3.2 玻璃与节能（安徽 · 21 层 · 夏热冬冷）

> 依据 **DB34/1466** 类安徽省标准；本项目 **21 层主动建筑**，南向窗墙比大，宜 **就高不就低**。

| 项目 | 通用倾向 | 备注 |
|------|----------|------|
| 构造 | **三玻两腔**（5+12A+5+12A+5）或 **双玻 Low-E + 暖边条** | 下单前出 **K 值检测报告** |
| 传热系数 K | 南向大窗：**≤ 1.8 W/(m²·K)** 倾向；北向：**≤ 2.0** | 21 层表 4.2.6-2 主动建筑限值随窗墙比变化 |
| Low-E | **高透或单银 Low-E**（南向采光） | 北向可用保温型膜系 |
| 暖边条 | **必须** | 14 号窗框结露 |
| 中空 | 12A 或 12A+12A；**充干燥空气或氩气** | — |
| 水密 / 气密 | 不低于国标 **6 级** 倾向（21 层风压） | 合同写清等级 |
| 抗风压 | **≥ 9 级** 倾向（21 层） | 厂家按洞口计算书 |

### 3.3 隔声（21 层）

| 项目 | 倾向 |
|------|------|
| 玻璃 | 三玻两腔 + 不同厚度 **不对称组合**（如 5+6） |
| 开启扇 | **密封胶条三元 EPDM**；关闭时多点压紧 |
| 目标 | 比原窗 **Rw 提升 ≥ 5 dB**（无实验室报告时合同写 **计权隔声 ≥ 32 dB** 倾向） |

详 [13-隔音系统 §7](13-隔音系统.md)。

### 3.4 防结露

| 项目 | 要求 |
|------|------|
| 窗框 | 断桥 + 暖边条 + 室内 **保温延伸线**（部分厂家自带） |
| 安装 | 框体与墙 **发泡 + 耐候密封胶**；内侧与 02 **隔汽层** 搭接（§6） |
| 窗台 | 内侧 **保温板/石材下口** 不形成冷桥（02 §14） |

详 [14-防潮防结露系统 §6–9](14-防潮防结露系统.md)。

## 4. 外窗策略 · 必新做 vs 检测评估（2026-06-28 甲方确认）

### 4.1 总体分工

| 类别 | 策略 | 说明 |
|------|------|------|
| **A · 必新做** | **直接按 §3.1 新做系统窗** | 现场 **目前无整樘外窗** 或拆改后 **新开口** |
| **B · 检测后定** | **先检测 → 再定保留/维修/打掉重做** | 毛坯 **原有外窗**；结论写入 [19-现场沟通记录](19-现场沟通记录.md) + 拍照 |

**统一型式**：无论 A/B，最终外窗均按 **§3.1 断桥铝上下分格**（B 类若保留旧窗则除外，须检测合格）。

### 4.2 A 类 · 必新做（2 处）

> 甲方口述：**「目前没有窗的」= 阳台 + 次卧。** 与 [22-拆改方案](22-拆改方案意向.md) 对照如下。

| 序号 | 甲方指称 | 与方案/模型对照 | 新做范围 | 施工要点 |
|------|----------|-----------------|----------|----------|
| **A1** | **阳台**（口述「南向阳台」） | 方案 E 为 **北阳台整体封闭进室**（洗衣区/客厅北缘）；02 注明 **无独立南阳台房间** | **北向原外立面 / 封闭阳台外缘** 上 **目前无整樘窗** 的洞口 **统一定做 §3.1** | ① 物业外立面 ② 02 **§4 外墙** 与窗框同步 ③ 结露 **🔴 高**（14）④ 封闭后进室须 **防水+找坡**（02 §26.1） |
| **A2** | **次卧窗** | 次卧 **北扩** 吃进原阳台进深（R35A/R31 已拆）；**靠北原外轮廓** 为新界面 | **次卧北向（及复尺确认的其他新洞口）新做 §3.1** | ① **FP AC2** 贴西墙 T18，窗勿挡 FP ② **FA2** 低位送风/北墙高位排风（08）③ **儿童安全锁** ④ 若 **南向旧窗 T3 仍存在** → 归 **B2** |

**A1 复尺时必做：** 明确口述「南向」与现场 **北封闭阳台** 是否同一区域；**以现场编号为准**，避免下单错误。

**A2 复尺时必做：** 区分 **北向新窗**（必做）与 **南向 T3 旧窗**（检测），勿混为一樘。

#### 4.3.3 检测结论 · 处置标准

| 结论 | 条件（满足任一即倾向重做） | 处置 |
|------|---------------------------|------|
| **保留** | 断桥 + 双玻及以上 + 密封良好 + 无严重结露 + 五金正常 | 仅做 **周边密封维修**、纱窗更换；与 02 **隔汽搭接** 检查 |
| **维修** | 型材尚可；局部密封/五金/玻璃问题 | 换胶条、五金、局部玻璃；**不解决 K 值则改重做** |

**21 层建议：** B 类南向大窗若无法提供 **K ≤ 1.8** 依据，倾向 **重做**。

### 4.4 施工顺序建议

```text
1. 物业报审（外立面、A 类新做 + B 类可能更换）
2. 全场外窗复尺 + B 类逐樘检测（表 4.3.1）→ 拍照
3. 确定 A1/A2 洞口 + B 类结论清单 → 统一下单（同色同系列）
4. B 类「保留」先维修；「重做」与 A 类同批安装
5. 安装 → 隔汽恢复 → 淋水 → 验收（§11）
```

## 6. 安装标准（索引 02）

### 6.1 换窗工序

```text
1. 物业审批 + 外立面色确认
2. 洞口复尺 → 下单（含 K 值、抗风压计算书）
3. 旧窗拆除 → 清理洞口 → 检查保温层/隔汽层破损
4. 窗框就位 → 水平垂直 → 膨胀螺栓/固定片
5. 框与墙：PU 发泡（不 overspray 室内）→ 内外耐候密封胶
6. 隔汽膜与窗框内侧 **搭接 + 胶带**（02 §4、§15）
7. 窗台保温/石材 → 室内收边
8. 淋水试验 + 气密检查 → 拍照
9. 纱窗、五金、儿童锁调试
```

### 6.2 节点要点

| 节点 | 做法 |
|------|------|
| 框墙间隙 | 发泡填充 **≥80%**；内外 **双道密封胶** |
| 隔汽层 | 室内侧膜 **绕过框体或贴框内侧** 连续；破口必补（21 §5.3） |
| 窗台 | 外坡水、内保温；**禁止** 窗台石直接坐冷混凝土无保温 |
| 下固定大玻璃 | 按厂家 **厚玻璃 + 垫块 + 结构胶** 要求；高层 **安全玻璃**（钢化/夹胶按计算） |
| 横梃 | 排水孔通畅；横梃下口 **室内侧** 不积水 |

## 7. 门系统

### 7.1 入户门（玄关）

| 项目 | 倾向 |
|------|------|
| 现状 | 左下入户，**内开**（22） |
| 换门 | 若更换：**断桥装甲门或钢木安全门**；**乙级及以上** 防盗倾向 |
| 密封 | 三道密封条；**隔声 ≥ 30 dB** 倾向 |
| 门槛 | 与 **地暖完成面** 衔接；**不形成冷桥**；室内侧可石材/不锈钢 |
| 弱电 | 玄关 T18 电箱在侧墙，**门扇不遮挡** |

### 7.2 室内门

| 用途 | 型式倾向 | 备注 |
|------|----------|------|
| 卧室 ×3 | 平开；门扇 **≥ 40 mm**；**磁吸/密封条** | 13：隔声 |
| 书房 | 同卧室或 **静音门** | 邻公区 |
| 主卫 / 客卫 | **防潮免漆** 或铝框玻璃；门套 **防潮** | 03：门槛 **±0 齐平**；门内 **-10～15 mm** |
| 洗衣区 ↔ 衣帽间 | **平开防水门** 倾向（22 待决） | 02 §26.2；门坎 **20–30 mm**（洗衣区有地漏） |
| 开放厨房 | **可关闭玻璃移门** 倾向（若燃气要求） | 22 §3；FA5 送风避灶台 |

### 7.3 门与地暖门槛（卫间）

> 决策 [20-009](20-问题决策记录.md)：**门槛石顶 = 干区 ±0**；门内低于干区 **10–15 mm**。

- 门套下口：**止水坎 + 门槛石** 一次成型考虑
- **地暖管尽量不穿门槛正下方**（03）

## 9. 施工要求

1. **先物业后下单**：外立面颜色、开启方式、是否允许外开
2. **复尺**：每樘 **宽×高×墙厚×横梃标高**；斜墙/飘窗单独量
3. **与墙体保温分工**：窗框安装 **在隔汽膜铺设之后、石膏板封板之前** 或按厂家节点 **恢复隔汽**
4. **与吊顶**：南窗 **FA1 排风** 吊顶留检修口；窗帘盒深度 **≥ 120 mm**
5. **成品保护**：安装后贴膜；地暖/墙施工 **禁止砸框**

---

## 10. 禁止事项

- 未获 **物业书面同意** 擅自换窗外立面
- **非断桥**、单层玻璃、无暖边条
- 下固定大窗 **无安全玻璃** 计算
- 框墙 **不发泡、不密封**；破坏隔汽 **不恢复**
- 开启扇与 **新风排风口** 位置冲突
- 卫间门 **无防潮处理**、门槛 **高于干区完成面**
- 背靠背 **插座** 在窗侧墙（02 §12）
