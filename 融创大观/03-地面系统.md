## 3. 完成面标高体系（2026-06-28 已确认）

### 3.1 基准

| 项目 | 规定 |
|------|------|
| **干区完成面** | 全屋统一 **±0**（客厅、卧室、书房、厨房干区、玄关、衣帽间、洗衣区、封闭阳台等） |
| **地面隔音** | **不做**（不设浮筑楼板、隔音垫、减振垫层） |

### 3.2 卫生间与干区关系

| 项目 | 规定 |
|------|------|
| **门槛石顶面** | 与干区完成面 **齐平**（同为 ±0） |
| **卫间门内地面** | 低于干区/门槛 **10～15 mm**（相对 ±0 为 **-10～-15 mm**） |
| **找坡方向** | 自门口/墙边 → **地漏** |
| **淋浴区坡度** | **略大**（倾向 **1.5～2%**） |
| **非淋浴区坡度** | **略小**（倾向 **1～1.5%**） |
| **门槛石下** | 必须做 **止水坎**；防水 **上翻并连续收口** |
| **地暖管** | **尽量不要从门槛石正下方穿过**；盘管绕开门槛/门洞下压区 |

```text
        干区 ±0 ─────────────── 门槛石顶 ±0
              │    止水坎+防水上翻
              │    ┌── 卫间 -10~-15mm（砖面）
              │    │     ↘ 找坡 → 地漏
              └────┘
```

### 3.3 地暖区典型抬升（干区无卫间双防水）

自 **结构楼板** 至 **干区 ±0** 完成面，倾向总厚度 **80～95 mm**（现场以弹线为准）：

| 层次 | 厚度倾向 |
|------|----------|
| 保温板 | **25 mm**（EPS，亳州 21 层通用倾向） |
| 反射膜 + 盘管 | 管外径 **16 mm** |
| 回填覆盖管顶 | **40 mm** |
| 饰面层（瓷砖） | **10 mm** |

**无地暖区**（玄关、衣帽间、洗衣区、封闭阳台）：完成面同样 **±0**，不设保温盘管，通过 **找厚找平** 与地暖区衔接。

### 3.4 与新风送风口

[08-新风系统](./08-新风系统.md) 低位送风完成面以上 **300 mm**；地暖回填后须复核送风口底标高，**避免被抬升后冲突**。

## 5. 施工顺序

### 5.1 干区（客厅/卧/书房/厨房 — 无卫间双防水）

```text
1. 结构基层清理、弹线（±0 基准）
2. 基层找平
3. 铺保温板 25 mm → 反射膜 → 墙脚边界保温条
4. 安装分集水器（设备带，高于盘管）
5. 回字形铺 PE-RT Ø16 → 卡钉 → 硅晶网
6. 接分集水器支路；主管接 [A] 二次侧
7. 试压 0.6 MPa / 30 min → 拍照
8. 带压回填（C15 细石或 1:3 砂浆，盖管顶 40 mm）
9. 养护 → 饰面层（瓷砖推荐 / 地暖专用地板）
10. 冲洗调试
```

### 5.2 卫生间（主卫 / 客卫 — 满铺地暖 + 双道防水）

**必须按以下顺序，禁止颠倒：**

```text
1. 基层找平 / 找坡（门口高、向地漏方向低，预留 -10～-15 mm 完成面差）
2. 止水坎（门槛石下）+ 第一道防水 + 闭水试验 → 拍照
3. 防水保护层（水泥砂浆或保护涂料，防扎破）
4. 铺保温板 → 反射膜 → 边界条
5. 地暖盘管（回字形，满铺；绕开门槛正下方）→ 卡钉 → 硅晶网
6. 带压回填（保持 0.6 MPa 监测）
7. 二次找坡（淋浴区坡度略大，非淋浴略小，仍向地漏）
8. 第二道防水 + 闭水试验 → 拍照
9. 铺砖（砖面完成面：门内 -10～-15 mm；门槛石顶 ±0）
```

| 阶段 | 要点 |
|------|------|
| 第一道防水 | 地面满涂 + 墙根上翻 **≥300 mm** + 门槛侧 **止水坎与防水连续** |
| 盘管 | **满铺**；**避开门槛石正下方** |
| 带压回填 | 试压 **0.6 MPa** 不泄压前提下回填 |
| 第二道防水 | 覆盖回填层；淋浴区 **1.5～2%**，干区卫间 **1～1.5%** |
| 闭水 | 每道防水后 **24～48 h** 闭水，楼下无渗漏 |

### 5.3 无地暖区

```text
基层找平 → 找厚至 ±0 → 饰面层（与相邻地暖区过渡顺平）
```

洗衣区、封闭阳台若需防水找坡，按 02 号厨卫墙节，**不走 §5.2 双道+地暖叠层**（无盘管）。

## 6. 地暖五路方案

> **SketchUp 自动画管已暂停**；以本文 + [`floor_heating.json`](../融创大观-3D/模型/全屋/方案推敲/floor_heating.json) 为准。

### 6.1 基本参数

| 项目 | 数值 |
|------|------|
| 管材 | PE-RT 阻氧，外径 **16 mm** |
| 盘管间距 | **200 mm**（8 SU） |
| 墙内退距 | **100 mm**（4 SU） |
| 铺管方式 | **回字形**（huizi），**禁止双管平行并排** |
| 单路长度 | 建议 **≤ 100 m** |
| 试压 | **0.6 MPa**，30 min，压降 ≤ **0.05 MPa** |

### 6.2 五路分区与干管

| 回路 | 覆盖 | 入口 | 干管要点 |
|------|------|------|----------|
| **FH1** | 客厅 | 客厅_南入口 | 分集水器 → y=42 → y=157 → 客厅；**不经客卫** |
| **FH2** | 次卧 | 次卧_南入口 | 分集水器 → 南排 → 次卧 |
| **FH3** | 书房 | 书房_南入口 | 分集水器 → 南排 → 书房 |
| **FH4** | 主卧 + **主卫满铺** | 主卧_南入口 | → 主卧；经「主卧_进主卫」进主卫 |
| **FH5** | 厨房 + **客卫满铺** | 厨房_南入口 | → 厨房；经「厨房_进客卫」进客卫（**仅盘管联通，干管禁止**） |

**干管禁区**：FH1 **禁止穿越客卫**；全局禁止经「厨房_进客卫」走 **干管**。

分集水器：设备带 `fh_manifold`，约 SU **(277, 42)**。

### 6.3 各房间净界（设计中1 · SU 坐标）

| 房间 | X | Y | 热侧 | 回路 | 估算盘管长 m |
|------|---|---|------|------|-------------|
| 客厅 | 93–228 | 157–307 | 北 | FH1 | ~50 |
| 次卧 | 238–345 | 157–307 | 北 | FH2 | ~35 |
| 书房 | 277–393 | 8–147 | 南 | FH3 | ~35 |
| 主卧 | 355–470 | 157–307 | 北 | FH4 | ~40 |
| **主卫** | 408–486 | 8–147 | 南 | FH4 | ~25 |
| 厨房 | 77–167 | 8–147 | 南 | FH5 | ~30 |
| **客卫** | 177–267 | 8–147 | 南 | FH5 | ~25 |

**FH4 合计 ~65 m、FH5 合计 ~55 m**，均 **< 100 m**。

### 6.4 回字形铺管（huizi）

每路为 **一根连续管** 闭式回路；矩形螺旋 → 远端 180° 折返 → 反向螺旋回程；**平面禁止交叉**。

示意：[`tools/huizi.svg`](../融创大观-3D/tools/huizi.svg)

**卫间附加要求**：

- **满铺**回字，淋浴区可 **间距加密至 150 mm**（待现场）
- 盘管 **绕开门槛正下方** 300 mm 宽条带
- 地漏、止水坎位置 **防水加强** 涂布

## 10. 禁止事项

- 干区完成面 **不统一 ±0**
- 卫间门槛高于或低于干区完成面（**必须齐平 ±0**）
- 卫间门内与干区 **无 10～15 mm 高差**
- 门槛下 **无止水坎** 或防水 **不连续**
- 地暖管 **从门槛石正下方穿过**
- **双管平行并排** 代替回字形
- FH1 干管穿客卫
- 卫间 **跳过双道防水** 或 **先做地暖后做第一道防水**
- 地面做 **浮筑隔音**
- 未闭水、未试压、未拍照即回填或贴砖
