> 最后更新：2026-06-28

## 1. 目标

- 定义地面完成面标高、保温、找平及门槛衔接原则
- 若做地暖：明确分路、干管走向、盘管方式、管径间距及验收要求
- 与空气能、冷热水、冷凝水、隔音系统联动，避免后期返工

全屋地面（含各房间、过道、卫生间），以及与分集水器、设备带相关的地暖干管区域。

### 3.3 盘管方式：回字形（huizi / rectangular_spiral）

本项目地暖采用 **回字形铺管**（亦称矩形螺旋、回字盘管），**不是**「双管平行并排」走法。

**原理示意**（见 `融创大观-3D/tools/huizi.svg`、`huizi_fixed.png`）：

```text
入口（热侧/分集水器来向）
  │
  ▼
┌─────────────────────────┐
│ →→→→→→→→→→→→→→→→→→→→ │  第 1 圈：贴墙根，矩形走线
│ ↓                     ↑ │
│ →→→→→→→→→→→→→→→→→→→→ │  第 2 圈：向内退一个管间距
│ ↓                     ↑ │
│   … 逐圈向中心收缩 …   │
│ ↓                     ↑ │
│      ╭─ U 折返 ─╮       │  最远端 180° 折返
│      ╰─────────╯       │
│ ←←←←←←←←←←←←←←←←←←←← │  回程：沿相邻内圈反向螺旋
└─────────────────────────┘
出口（同一路回分集水器）
```

**说明**：每路地暖（FH1–FH5）是 **一根连续加热管** 形成的闭合回路——从分集水器供水口出、回水口回；盘管在房间内呈 **回字矩形螺旋**，而非两根管并排蛇形。

**必须满足**：

1. **单管回路**，供、回在分集水器上为同一路的两个接口
2. 从入口沿墙根 **矩形螺旋向中心** 铺设，间距 200 mm
3. 最远端 **180° 折返**，沿相邻平行圈 **反向螺旋** 回到出口
4. 平面内 **禁止管道交叉**（放线时注意中心区域）
5. **禁止穿墙**；转弯在房间净界内，墙内退距 100 mm
6. 必须有完整 **回程**，不能「只去不回」

**禁止的做法**：

| 做法 | 问题 |
|------|------|
| **双管平行并排**（parallel_pair） | 非本项目标准；易与回字混淆 |
| 单根蛇形 + 侧边竖向折返 | 竖向段横穿各圈横管，视觉交叉 |
| 无回程的单向铺管 | 非完整回路，无法接分集水器 |
| 穿墙、越房间净界 | 检修困难，违反隐蔽工程要求 |
| 干管未经确认穿越客卫（FH1） | 已明确禁止 |

**施工/放线原则**：

- 先弹房间净界、入口位置与 200 mm 间距控制线
- 从 **热侧**（靠分集水器/干管来向一侧）进入
- 逐段正交放线，转弯半径符合管材要求（通常 ≥ 5D，D 为管外径）
- 单路盘管长度建议 ≤ 100 m（待热负荷复核）

## 4. 系统设备（热源侧）

本项目地暖热源来自 **空气能两联供二次侧**，与生活热水系统分离（见 [09-空气能及水系统](./09-空气能及水系统.md)）。

| 设备 | 作用 | 本项目配置 |
|------|------|------------|
| 空气能热泵主机 | 提供采暖热水 | 厨房后北侧平台，型号待选型 |
| 缓冲水箱 / 耦合罐 | 稳定二次侧水力，与 FP 并联 | 容积待算 |
| 地暖循环泵 | 驱动地暖回路循环 | 常含于主机/水箱系统 |
| 地暖分集水器 | 5 路流量分配与汇集；排气、泄水 | 设备带，5 路（FH1–FH5） |
| Y 型过滤器 | 过滤杂质，保护阀门与盘管 | 装于地暖主管 |
| 温控器 + 电热执行器 | 分室（分区）自动调温 | **5 块一体温控器**，与 FP 同区联动，见 [07-风机盘管系统 §6](./07-风机盘管系统.md) |

**水力关系**：

```text
缓冲水箱二次出水 → 分集水器总供水口
分集水器总回水口 → 缓冲水箱二次回水
分集水器各支路 ↔ 各房间回字盘管（单管闭式回路）
```

风机盘管与地暖 **并联** 于二次侧，不是串联。

## 5. 地面辅材与回填

| 类别 | 材料 | 规格倾向 | 说明 |
|------|------|----------|------|
| 绝热 | 保温板 | EPS 或 XPS，厚 20–30 mm | 厚度待热工计算；**仅用于地面**，与外墙禁 XPS 原则不冲突 |
| 反射 | 铝箔反射膜 | 带施工网格 | 铺于保温板之上 |
| 边界 | 边界保温条 | 高 ≥ 保温板厚 | 沿内墙脚整圈，吸收回填胀缩 |
| 固定 | 盘管卡钉 | 间距 500–800 mm | 固定 PE-RT 管，避免上浮 |
| 加固 | 硅晶网 / 钢丝网 | 网格约 50×50 mm | 盘管上方，配合回填防裂 |
| 伸缩 | HDPE 伸缩缝板 | 宽 8–12 mm | 大面积或超长房间设置 |
| 回填 | 细石混凝土 / 水泥砂浆 | C15 细石或 1:3 砂浆 | 覆盖管顶 30–50 mm，待设计 |

详细清单见 [`物料清单_地暖系统.md`](../融创大观-3D/物料/清单/物料清单_地暖系统.md)。

## 6. 标准做法

### 6.1 施工顺序（地暖部分）

```text
1. 基层清理、找平验收
2. 弹线（分集水器位置、盘管间距、伸缩缝）
3. 铺保温板 → 贴反射膜 → 墙脚边界保温条
4. 安装分集水器（高于盘管，带排气/泄水）
5. 铺地暖管（回字形）→ 卡钉固定 → 铺硅晶网
6. 连接分集水器 ↔ 各支路；连接主管 ↔ 缓冲水箱
7. 水压试验（保压记录）→ 拍照留档
8. 回填找平层 → 养护 → 面层（瓷砖/地板等）
9. 系统冲洗 → 初次运行调试
```

### 6.2 与地面其他层关系

| 层次（自下而上） | 内容 |
|------------------|------|
| 结构楼板 | 原建筑 |
| 绝热层 | 保温板 + 反射膜 |
| 盘管层 | PE-RT 16 mm，Z 示意 2.8 SU |
| 回填层 | 细石混凝土 / 砂浆 |
| 饰面层 | 瓷砖（热阻小，推荐）或地暖专用地板 |

### 6.3 其他

- 门槛石与完成面标高：待现场复尺（地暖+回填会抬升地面）
- 冷凝水、生活水管与地暖分层施工，参见各系统文档

## 8. 施工要求

- 分集水器位置固定、标高正确后再放线盘管
- 每路独立试压，保压 30 min 无渗漏后再回填
- 盘管间距、墙内退距、转弯半径符合管材说明
- 卫生间等区域是否满铺地暖需单独确认（FH4/FH5 含卫间）
- 单路盘管长度建议 ≤ 100 m
- 大面积地面设伸缩缝

---

## 9. 禁止事项

- 采用 **双管平行并排** 代替回字形（非本项目标准）
- 盘管平面交叉、穿承重墙未预留套管
- 干管未经确认穿越客卫（FH1 已明确禁止）
- 无回程、不完整回路
- 封闭回填层前未拍照留档（见 21 号文档）
- 使用普通泡沫地垫代替地暖专用界面材料（若铺木地板）

---

## 10. 隐蔽验收

地暖铺设后、回填前至少检查：

- [ ] 五路分区与分集水器对应关系
- [ ] 回字形走线、管间距 200 mm、墙内退距
- [ ] 无交叉、无死弯、卡钉牢固
- [ ] 边界保温条连续、反射膜搭接
- [ ] 水压试验记录
- [ ] 全景 + 分路特写照片（见 21 号文档 4.2 节）
