# 02 · 墙体系统

> 最后更新：2026-06-28  
> 文档状态：v0.1 · 工程手册级别（待现场复尺后迭代）

---

## 1. 目标

墙体系统不是「砌墙 + 批腻子」，而是同时解决：

```text
保温 + 隔音 + 防结露 + 防火 + 可维修 + 可验收 + 可拍照留档 + 设备挂装
```

**不做会怎样：**

- 冬季外墙内表面结露 → 发霉、墙皮鼓包、柜体背后潮湿
- 隔汽层缺失或不连续 → 岩棉吸潮失效、保温性能下降
- 冷桥未处理 → 插座、穿墙管、窗框周边结露
- 设备墙未加固 → 风机盘管、电视、浴室柜脱落
- 封板前未拍照 → 后期无法定位加固点、无法追责

---

## 2. 适用范围

适用于本项目全部墙体相关施工，包括：

- 六类墙体的分类设计与施工
- 外墙内侧保温隔音系统
- 设备墙欧松板加固
- 插座盒、穿墙、飘窗等节点
- 与新风、冷凝水、空调水管、电气的穿墙交叉

**不适用：** 外立面改造（需物业及专业确认）

## 3. 墙体分类

全屋墙体按功能分为六类，**禁止混用施工标准**：

| 类型 | 功能 | 保温 | 隔音 | 适用位置 |
|------|------|------|------|----------|
| **外墙** | 保温 + 隔音 + 防结露 | 是 | 是 | 主卧、次卧、客厅、阳台、飘窗、靠室外墙面 |
| **分户墙** | 隔音 | 否 | 是 | 与邻居相邻的墙体 |
| **电梯 / 楼梯墙** | 保温 + 隔音 | 是 | 是 | 靠电梯井、楼梯间的冷墙面 |
| **室内隔墙** | 普通分隔 | 否 | 可选 | 房间内部非承重分隔墙 |
| **厨卫墙** | 防水、挂重物 | 否 | 可选 | 厨房、卫生间墙体 |
| **设备墙** | 管线、风机盘管、控制器、挂装设备 | 否 | 是 | 放置或隐藏设备的墙体 |

### 3.1 分类原则

- 一面墙可能同时具有多种功能（如靠外的设备墙），按**最高要求**执行
- 靠电梯井 / 楼梯间的墙面，即使不是传统「外墙」，也按**电梯/楼梯墙**标准做保温隔音
- 设备墙隔音要求：减少设备噪音对相邻房间的影响

#### 3.4.1 总推荐（厚度倾向）

| 墙体位置 | 推荐施工体系 | 厚度倾向 | 重点目标 |
|----------|--------------|----------|----------|
| 南向窗带 / 落地窗侧边 | 100mm 加气块/ALC + 局部欧松板；噪音大时内侧加 50 岩棉 + 双层石膏板 | 100–150mm | 收口稳、防裂、窗帘盒/柜体固定 |
| 室内普通墙 | **75 轻钢龙骨 + 50 岩棉 + 双面双层 12mm 石膏板**；门洞/柜体处改 100mm 轻质砖 | 90–120mm | 分隔、隔音、施工快 |
| 北向封闭阳台边界 | **半外墙加强**：保温 + 防潮 + 防结露（见 3.4.4） | 120–180mm | 防冷桥、防霉、洗衣区防潮 |
| 衣帽间新建围合墙 | **50/75 轻钢龙骨薄墙 + 18mm 欧松板**（推荐 90–100mm 版） | 75–100mm | 薄、直、柜体/门套稳定 |

**倾向（与项目原则一致）：**

- 北/南边界：**不要太薄**
- 室内普通墙：**可轻钢龙骨**
- 衣帽间：**可薄，但必须欧松板加固**

#### 3.4.2 ① 南向窗带 / 半外墙边界墙

**适用：** 客厅 R05、次卧 R06B、书房 R08、主卧 R10C 等 **南向玻璃墙两侧及窗下普通墙带**；落地窗收边、窗帘盒、电动窗帘、新风/空调穿墙。

**推荐 A · 稳妥型（优先）**

```text
室内侧
  ↓ 乳胶漆 / 饰面
  ↓ 腻子
  ↓ 12mm 石膏板（或水泥纤维板）
  ↓ 局部 18mm 欧松板（窗帘盒、侧柜、吊柜）
  ↓ 100mm 加气混凝土砌块 / ALC 轻质条板
  ↓ 抹灰找平 + 抗裂网
  ↓ 外侧收边 / 窗框节点
```

**推荐 B · 隔音保温加强型**（外部噪音明显时）

在 A 基础上，室内侧再加：`50mm 憎水岩棉 + 75 龙骨 + 双层 12mm 石膏板错缝`（总厚度增加，舒适度更好）。

**材料标准：** 加气块 GB/T 11968；轻质条板 GB/T 23451；岩棉/燃烧性能 GB 8624 **A 级不燃**（见 §10.2，禁止杂牌泡沫板）。

**禁止：** 单层 9.5mm 石膏板薄隔墙做窗侧边；穿墙孔留空（见 §13）。

#### 3.4.3 ② 室内普通墙

**适用：** 书房新建白墙（§27.3.2）、次卧新门洞隔墙（§27.3.4）、卧室/书房/通道之间分隔。

**方案 1 · 轻钢龙骨（默认推荐）**

```text
房间 A
  ↓ 乳胶漆
  ↓ 腻子
  ↓ 12mm 石膏板
  ↓ 12mm 石膏板（错缝）
  ↓ 75 轻钢龙骨 + 减振垫
  ↓ 50mm 憎水岩棉（填满）
  ↓ 12mm 石膏板 ×2（错缝）
  ↓ 房间 B
```

厚度约 **120mm**。龙骨 GB/T 11981；石膏板 GB/T 9775（见 §10.3、§10.4）。

**方案 2 · 100mm 轻质砖/ALC**（门洞多、要实体感、玄关/厨房收口）

```text
房间 A → 抹灰/乳胶漆 → 100mm 加气块/ALC → 抹灰/乳胶漆 → 房间 B
```

**选用建议：**

| 位置 | 推荐 |
|------|------|
| 书房扩大、卧室之间 | 方案 1 |
| 玄关/厨房边界、门洞密集、柜体侧墙 | 方案 2 或方案 1 + 局部欧松板 |

**禁止：** `75 龙骨 + 单层 9.5mm 石膏板 + 不填岩棉`（§10.4）。

#### 3.4.4 ③ 北向封闭阳台边界墙（半外墙加强）

**适用：** §26.1 整条北侧原阳台进室——洗衣区、客厅北缘、次卧北扩；模型分区「阳台」「洗衣区」及客厅/次卧靠北段。

**比南边界更严：** 北向日照少、冬季更冷、潮气易积；柜体背后、窗边、洗衣机附近重点防潮（见 [14-防潮防结露系统](./14-防潮防结露系统.md)）。

**情形 A · 贴在原结构外轮廓上（原阳台外立面、栏板内侧）**

→ **必须按 [§4 外墙系统](#4-外墙系统) 全做**（隔汽膜 + 减振垫 + 75 龙骨 + 50mm 憎水岩棉 + 双层 12mm 石膏板错缝）。  
与用户资料「100mm 轻质体 + 50 岩棉 + 防潮石膏板」**不矛盾**：这里是**结构外墙饰面**，不是新建填充墙。

**情形 B · 封闭后进室内新建的竖向/横向分隔墙**（如洗衣区与客厅之间、次卧北扩后的新隔断）

```text
室内侧
  ↓ 乳胶漆
  ↓ 耐水腻子
  ↓ 双层 12mm 石膏板 / 防潮石膏板（错缝）
  ↓ 局部 18mm 欧松板
  ↓ 75 轻钢龙骨 + 减振垫
  ↓ 50mm 憎水岩棉（填满）
  ↓ 100mm 加气块 / ALC（新建墙体主体）
  ↓ 北向/湿区侧饰面
```

空间紧张时最低限度：`防潮石膏板 + 50 龙骨 + 40–50mm 憎水岩棉 + 原墙/100mm 轻质墙`，但**洗衣区、地漏附近不得再减**。

**墙脚（洗衣区 / 生活阳台）：**

```text
地面防水上翻 ≥300mm + 墙脚水泥基防水 + 耐水腻子
```

**柜体背后：** 留 10–20mm 通风缝；背板勿完全封死；§26.2.4 要求 20–30mm 透气缝或防潮膜。

#### 3.4.5 ④ 衣帽间新建围合墙（薄墙）

**适用：** 面向玄关/走廊的新建堵墙（§26.2.3）；**不含**东/西分户墙、北/南外墙。

| 方案 | 构造 | 厚度 | 适用 |
|------|------|------|------|
| A 极限薄 | 50 龙骨 + 40/50 岩棉 + 双面 12mm 石膏板 | ~75mm | 仅轻分隔、不挂重 |
| **B 推荐** | 50 龙骨 + 40/50 岩棉 + 双面 12mm 石膏板 + **整面或局部 18mm 欧松板** | **90–100mm** | 柜体、门套、挂衣杆、灯带、插座 |
| C 隔音加强 | 75 龙骨 + 50 岩棉 + 双面双层石膏板 + 欧松板 | 110–130mm | 衣帽间贴卧室、要减噪 |

**推荐选 B。** 固定点欧松板范围超出锚点 **≥100mm**，封板前拍照（§11、§21）。

#### 3.4.6 拆改后各区域 → 四类对照

| 区域 / 动作 | 新建墙类型 | 章节 |
|-------------|------------|------|
| 客厅/次卧/主卧 **南向窗侧边** | ① 半外墙边界 | 3.4.2 |
| 书房扩大、次卧门洞新墙 | ② 室内普通（书房建议方案 1 隔音加强） | 3.4.3、§27.3.2 |
| **北阳台**原外立面 | ③ 情形 A → §4 外墙 | 3.4.4、§26.1 |
| 洗衣区周边 **新建**隔断 | ③ 情形 B + §8 厨卫墙 | 3.4.4、§26.2 |
| 洗衣区 ↔ 衣帽间 **水平隔断** | §8 厨卫墙 + 门（非四类简单替代） | §26.2.2 |
| 衣帽间 **面向走廊堵墙** | ④ 衣帽间薄墙方案 B | 3.4.5、§26.2.3 |
| 衣帽间 **背面** | 不新建；结构墙做 §4 | §26.2.4 |
| 开放厨房剩余墙 / 吊柜墙 | §8 厨卫 + §9 设备 + 欧松板 | §27.3.1 |
| 主卫改门洞 | §8 厨卫墙 | §27.3.3 |

#### 3.4.7 封板前必拍（与 §21 一致）

- [ ] 龙骨间距、减振垫、门洞过梁/加固
- [ ] 岩棉填满、憎水型标识
- [ ] 欧松板位置与尺寸标注
- [ ] 插座盒、穿墙套管、发泡 + 密封胶
- [ ] 北阳台/洗衣区墙脚防水、找坡
- [ ] 隔汽层连续（贴结构外墙时）
- [ ] 管线编号与走向

#### 3.4.8 可发给施工方的摘要

> 拆改后新建墙分四类：**①南向窗带边界** 100mm 轻质砖/ALC + 窗边欧松板；**②室内普通** 75 龙骨 + 50 岩棉 + 双面双层 12mm 石膏板，门洞/柜体处改轻质砖或加固；**③北向封闭阳台** 原外立面按外墙全做，新建隔断按半外墙加强 + 墙脚防水；**④衣帽间** 50/75 龙骨薄墙 90–100mm + 18mm 欧松板。  
> 禁止：单层 9.5mm 石膏板、不填岩棉、穿墙孔留空、封板后开洞破坏防潮气密层。  
> 东/西分户墙只做隔音，不按外墙保温。

## 4. 外墙系统

### 4.1 目标

外墙系统重点不是简单贴保温材料，而是同时解决：

- 保温
- 隔音
- 防结露
- 防火
- 可维修
- 可验收
- 可拍照留档

### 4.2 适用位置

- **南、北外轮廓 + 北阳台外缘**（见 [3.2.1 节](#321-四向围护墙认定设计中1-模型)）——**不含东、西分户墙**
- 主卧、次卧、客厅、书房、厨房等**靠室外**的墙面段
- 封闭阳台原外立面、栏板内侧
- 飘窗两侧及顶部
- 靠电梯井 / 楼梯间的冷墙面（按电梯/楼梯墙标准，见第 6 节）

### 4.3 标准剖面（暂定）

由外向内：

```text
室外
  ↓
结构墙
  ↓
界面剂
  ↓
PE 隔汽膜，约 0.2mm
  ↓
减振垫
  ↓
75 轻钢龙骨
  ↓
50mm 憎水岩棉，约 60kg/m³
  ↓
局部 18mm 欧松板（挂装加固位置）
  ↓
第一层 12mm 石膏板
  ↓
阻尼胶 / 隔音阻尼层
  ↓
第二层 12mm 石膏板（与第一层错缝）
  ↓
腻子
  ↓
乳胶漆
  ↓
室内
```

### 4.4 为什么是这个顺序

| 层次 | 作用 | 不做会怎样 |
|------|------|------------|
| 界面剂 | 提高基层附着力 | 后续材料空鼓、脱落 |
| PE 隔汽膜 | 阻止室内水蒸气进入保温层 | 岩棉吸潮、保温失效、内侧结露 |
| 减振垫 | 阻断龙骨刚性声桥 | 隔音效果大打折扣 |
| 轻钢龙骨 + 岩棉 | 保温 + 填充隔音 | 冷热桥、噪音渗透 |
| 欧松板（局部） | 挂装设备基层 | 重物固定不牢 |
| 双层石膏板 + 阻尼层 | 质量隔声 | 隔音不达标 |
| 错缝 | 避免裂缝贯通 | 开裂、漏音 |

### 4.5 露点位置控制

```text
室内（暖侧）
  ↓
隔汽层 ← 露点应在此层外侧（保温层内）
  ↓
保温层（岩棉）
  ↓
结构墙（冷侧）
  ↓
室外
```

**关键：** 隔汽层必须连续、无破损；露点不应出现在室内侧饰面或结构墙室内表面。

## 7. 室内隔墙

### 7.1 目标

房间内部普通分隔：空间划分、基本隔音、门洞稳定、可走管线。**无保温要求**（除 [§3.4.3](#343--室内普通墙) 书房等需隔音升级处）。

### 7.2 推荐做法（默认）

见 **[§3.4.3 室内普通墙](#343--室内普通墙)**：

```text
75 轻钢龙骨 + 减振垫
  ↓
50mm 憎水岩棉（填满）
  ↓
双面双层 12mm 石膏板（错缝）+ 阻尼层
  ↓
腻子 → 乳胶漆
```

厚度约 **120mm**。

### 7.3 选用与升级

| 场景 | 做法 |
|------|------|
| 卧室、书房、通道普通分隔 | §3.4.3 方案 1 |
| 门洞多、柜体侧墙、玄关/厨房收口 | §3.4.3 方案 2（100mm 轻质砖/ALC）或方案 1 + 欧松板 |
| 书房扩大、邻公区/走廊 | §27.3.2：双层石膏板 + 岩棉（隔音升级） |
| 衣帽间围合 | §3.4.5 薄墙方案 B（90–100mm + 欧松板） |

### 7.4 要求

- 门洞上方过梁或加固龙骨；挂重物、门套位置 **18mm 欧松板**（§11）
- 有管线经过的隔墙，封板前定位并拍照
- **禁止** 单层 9.5mm 石膏板、空腔不填岩棉（§10.4）

## 8. 厨卫墙

### 8.1 目标

防水、防霉、可挂重物（浴室柜、厨房吊柜、壁挂马桶等）。

### 8.2 推荐做法

- 湿区必须做防水（高度、范围见 [15-施工顺序及隐蔽验收](./15-施工顺序及隐蔽验收.md)，待深化）
- 挂重物位置必须欧松板加固（见第 11 节）
- 瓷砖铺贴前防水必须验收 + 拍照

### 8.3 禁止

- 未做防水直接贴砖
- 壁挂马桶、浴室柜、吊柜不做基层加固

## 9. 设备墙

### 9.1 目标

承载管线、风机盘管、控制器及各类挂装设备，同时兼顾隔音。

### 9.2 适用位置

- 风机盘管安装墙
- 电视墙
- 热水器安装墙
- 新风控制器墙
- 弱电箱墙
- 网络设备墙
- 其他有隐藏管线或设备的墙体

### 9.3 做法

- 墙体骨架按实际需求（通常为轻钢龙骨）
- **挂装位置必须预埋 18mm 欧松板**（见第 11 节）
- 管线区域预留检修空间
- 与新风、冷凝水、空调水管、电气协调定位

## 10. 材料要求

### 10.1 隔汽膜

| 项目 | 要求 |
|------|------|
| 材质 | PE 隔汽膜，约 0.2mm |
| 搭接 | ≥ 100mm |
| 密封 | 专用胶带密封，禁止破损 |
| 穿孔 | 穿孔后必须补强密封 |
| 禁止 | 省略隔汽层；用普通塑料布替代 |

**验收看什么：** 整体连续性、搭接宽度、胶带密封、有无破损未补

**封闭前拍什么：** 整体铺设全景、搭接细节、穿孔补强（见 [21-拍照留档规则](./21-拍照留档规则.md)）

### 10.2 岩棉

| 项目 | 要求 |
|------|------|
| 厚度 | 50mm |
| 密度 | 约 60kg/m³ |
| 类型 | **憎水型** |
| 防火 | A 级 |
| 禁止 | 普通吸音棉、普通玻璃棉、低密度岩棉替代 |

**验收看什么：** 厚度、填充密实度、憎水型标识、无遗漏空腔

**不做会怎样：** 吸潮后保温失效、自重增加导致下垂、防火不达标

### 10.3 龙骨

| 项目 | 要求 |
|------|------|
| 规格 | 75 系列轻钢龙骨 |
| 镀锌 | 厚度建议 ≥ 0.6mm |
| 安装 | 配合减振垫，避免刚性声桥 |
| 禁止 | 过薄龙骨、非镀锌龙骨 |

**验收看什么：** 规格、镀锌层、间距、减振垫是否到位

### 10.4 石膏板

| 项目 | 要求 |
|------|------|
| 厚度 | 12mm + 12mm 双层 |
| 施工 | **错缝施工** |
| 中间层 | 阻尼胶 / 隔音阻尼层 |
| 禁止 | 单层 9.5mm 石膏板替代（外墙系统） |

**验收看什么：** 双层确认、错缝位置、阻尼层是否连续

### 10.5 欧松板

| 项目 | 要求 |
|------|------|
| 厚度 | 18mm |
| 用途 | 局部挂重物加固 |
| 范围 | 固定点四周至少外扩 100mm |
| 禁止 | 封板后凭记忆找位置 |

### 10.6 嵌入式暗装踢脚线（含灯带）

> **归属**：墙脚节点 → **本文档（02）**；完成面收口 → [03-地面系统 §9](./03-地面系统.md)；灯带电源 → [12-电气弱电系统 §4.7](./12-电气弱电系统.md)

**已确认（2026-06-28）**：干区采用 **嵌入式 / 暗装踢脚线**，**带灯带**（非明贴 PVC/木踢脚）。

| 项目 | 要求 |
|------|------|
| 形式 | **暗装**：踢脚型材嵌入墙脚凹槽，外立面与墙面 **平齐或微凹**，禁止凸出明装条 |
| 灯带 | 踢脚 **上沿内嵌 LED 灯带**，向上洗墙；色温倾向 **3000 K** 暖光（可调） |
| 适用区域 | **干区全屋**（客厅、卧、书房、厨房干区、玄关、衣帽间、封闭阳台等） |
| 卫间 | **主卫、客卫不做灯带踢脚**；墙脚用 **瓷砖踢脚线** 或防水收口，与 [03 §3.2](./03-地面系统.md) 门槛体系一致 |
| 高度倾向 | 踢脚可视高度 **40～60 mm**；含灯槽总深度 **≤ 25 mm**（留足墙脚凹槽） |

**墙脚构造示意（由外向内）**：

```text
完成面 ±0（地砖/地板）
  ↑
踢脚型材（暗装）+ LED 灯带 + 扩散罩
  ↑
墙脚预留凹槽（封板前弹出位置线）
  ↓
石膏板 / 腻子面层（与踢脚槽口齐平）
  ↓
龙骨 + 墙体主体
```

**施工要点**：

1. **封板前** 弹踢脚线走向；墙根留 **凹槽/留槽**（深度以型材为准），**边界保温条** 不得被踢脚槽破坏连续性
2. **地暖区**：踢脚安装在 **饰面层完成之后**；不钉穿 PE-RT 管；盘管距墙 100 mm 退距与踢脚槽 **兼容**
3. **灯带线管**：墙脚 **距地 50～150 mm** 预埋 **16 线管** 至检修盒/驱动位置；与插座回路 **分管**
4. **驱动**：24 V 恒压驱动，藏 **检修口 / 玄关柜 / 弱电箱附近**；须可更换
5. **与新风**：低位送风口 **300 mm** 处不得被踢脚灯带遮挡出风

**禁止**：

- 明装凸出踢脚代替暗装型材
- 卫间湿区使用无防护灯带
- 灯带无检修路径、驱动封死在不可拆结构内
- 未预埋线管即封墙贴踢脚

## 12. 插座盒节点

### 12.1 风险

插座盒容易造成：

- 漏风
- 漏音
- 隔汽层破坏
- 冷桥
- 结露
- 后期开裂

### 12.2 标准做法

| 要求 | 说明 |
|------|------|
| 禁止背靠背插座 | 同一位置内外墙两侧不得对称布置 |
| 错位 | 相邻墙面两侧插座建议错位 ≥ 300mm |
| 密封 | 插座盒周围发泡胶 + 密封胶处理 |
| 外墙插座 | 必须特别注意气密性，破坏隔汽层处必须补强 |
| 拍照 | 封板前拍照 |

### 12.3 验收看什么

- 是否背靠背
- 发泡胶、密封胶是否到位
- 隔汽层破口是否补强

## 13. 穿墙节点

### 13.1 适用范围

- 新风管
- 排气管
- 冷凝水管
- 水管（冷热水）
- 空调水管
- 电线管
- 其他穿墙管线

### 13.2 标准做法

```text
预留套管
  ↓
管道穿套管
  ↓
缝隙发泡胶填充
  ↓
内外密封胶收口
  ↓
饰面恢复
```

### 13.3 禁止

| 禁止 | 原因 |
|------|------|
| 直接打洞穿管 | 破坏结构、隔汽层 |
| 管周留空 | 漏风、漏音、进虫 |
| 只用水泥砂浆随便糊 | 不气密、易开裂 |
| 破坏隔汽层后不修补 | 结露、发霉 |
| 封闭前不拍照 | 无法追溯 |

### 13.4 验收看什么

- 套管规格是否合适
- 发泡胶是否填满
- 内外密封胶是否到位
- 隔汽层是否恢复

---

## 14. 飘窗节点

### 14.1 风险

飘窗是**结露高发区**，重点关注：

- 飘窗顶部
- 飘窗两侧
- 窗框周边
- 窗台石底部
- 飘窗柜背后
- 北向房间飘窗（风险更高）

### 14.2 建议做法（待现场确认后调整）

| 部位 | 建议 |
|------|------|
| 飘窗顶部 | 可考虑 20mm XPS + 水泥板（局部使用，非全屋 XPS 替代） |
| 飘窗两侧 | 按外墙系统处理 |
| 窗台石底部 | 增加保温处理 |
| 窗框周边 | 发泡 + 密封胶，切断冷桥 |
| 飘窗柜背后 | 预留通风或保温，避免柜体长期潮湿 |

### 14.3 不做会怎样

- 冬季结露 → 墙皮鼓包、发霉
- 柜体背后潮湿 → 板材变形、霉变

## 15. 冷桥处理

### 15.1 常见冷桥位置

- 窗框与墙体连接处
- 穿墙套管
- 插座盒
- 龙骨直接贴结构墙（未加减振垫 / 断桥）
- 飘窗顶部和底部
- 阳台门套周边

### 15.2 处理原则

```text
找到冷桥 → 切断或包裹 → 恢复隔汽层连续性 → 验收 → 拍照
```

### 15.3 做法

- 窗框：发泡 + 密封胶，窗框内侧保温延伸
- 穿墙管：套管 + 发泡 + 密封，隔汽层绕过或包裹套管
- 插座：发泡 + 密封，禁止背靠背
- 龙骨：减振垫隔断刚性连接

## 16. 防结露风险

### 16.1 本项目重点风险位置

| 位置 | 风险原因 | 应对 |
|------|----------|------|
| 北向房间外墙 | 温差大 | 严格按外墙剖面，隔汽层连续 |
| 飘窗 | 多面暴露 | 见第 14 节 |
| 窗框 | 金属 / PVC 导热 | 发泡 + 密封 + 内侧保温延伸 |
| 风口周边 | 冷气流 | 与 [08-新风系统](./08-新风系统.md) 联动 |
| 柜体背后 | 通风差 | 预留间隙或防潮处理 |
| 设备背后 | 热桥 / 冷桥 | 设备墙留通风或保温 |
| **封闭阳台北墙（方案 E）** | 原外立面变内墙，温差大 | 严格按外墙剖面；地面防水 + 找坡 |
| **家政模块（洗衣 + 衣帽间）** | 湿区相邻干区 | 见第 26.2 节；实墙 + 门，禁止柜体隔断 |

### 16.2 与防潮防结露系统的关系

详细露点计算、隔汽层设计见 [14-防潮防结露系统](./14-防潮防结露系统.md)（**v1.0 已深化**）。

## 17. 施工顺序

### 17.1 外墙 / 电梯墙 / 楼梯墙

```text
1. 现场复尺，确认墙面范围
2. 设备、插座、穿墙位置定位（甲方确认）
3. 结构墙界面剂
4. PE 隔汽膜铺设（搭接、密封）→ 验收 → 拍照
5. 减振垫 + 75 龙骨安装 → 验收 → 拍照
6. 50mm 憎水岩棉填充 → 验收 → 拍照
7. 欧松板加固（按设备清单）→ 验收 → 拍照（带卷尺）
8. 穿墙套管预留 → 验收 → 拍照
9. 第一层 12mm 石膏板
10. 阻尼胶 / 隔音阻尼层
11. 第二层 12mm 石膏板（错缝）
12. 腻子 + 乳胶漆（或后续饰面）
```

### 17.2 与其他系统的交叉顺序

| 交叉系统 | 顺序要求 |
|----------|----------|
| 水电 | 线管、底盒在封板前完成 |
| 新风 / 空调水管 / 冷凝水 | 穿墙套管在封板前预留 |
| 风机盘管 | 设备墙加固在封板前完成 |
| 弱电 | 底盒、线管在封板前完成 |

**原则：所有穿墙、加固、管线必须在石膏板封板前完成并验收拍照。**

## 18. 与其他系统的冲突点

| 冲突 | 说明 | 处理 |
|------|------|------|
| 新风穿墙 vs 隔汽层 | 打洞破坏隔汽层 | 套管 + 发泡 + 密封 + 隔汽层补强 |
| 风机盘管 vs 设备墙 | 需加固 + 留检修 | 提前定位，欧松板 + 检修空间 |
| 插座 vs 隔汽层 | 盒体破坏气密性 | 发泡 + 密封，禁止背靠背 |
| 柜体 vs 保温层 | 柜体贴墙可能形成冷桥 | 柜体背后留间隙或保温 |
| 拆改 vs 墙体分类 | 拆改后墙面类型变化 | 拆改确认后重新分类 |

## 19. 禁止事项

| # | 禁止 | 原因 |
|---|------|------|
| 1 | 外墙不做隔汽层 | 结露、发霉 |
| 2 | 用 XPS 作为内保温主体 | 防火、气密、结露风险 |
| 3 | 用普通玻璃棉 / 低密度岩棉替代憎水岩棉 | 吸潮、防火不达标 |
| 4 | 单层 9.5mm 石膏板做外墙封板 | 隔音、防火不达标 |
| 5 | 双层石膏板不错缝 | 裂缝贯通、漏音 |
| 6 | 背靠背插座 | 漏风、冷桥、结露 |
| 7 | 直接打洞穿管不用套管 | 破坏隔汽层 |
| 8 | 欧松板加固不拍照 | 后期无法定位 |
| 9 | 封板前未验收 | 隐患不可查 |
| 10 | 未确认设备位置就封板 | 返工成本高 |

## 26. 拆改方案 E · 专项墙体节点

> 对应 [22-拆改方案意向](./22-拆改方案意向.md) 方案 E  
> 状态：意向方案 · **未结构确认，不得施工**

### 26.1 封闭阳台（整体封闭）

#### 26.1.1 适用范围

北侧整条阳台带整体进室内，涉及：洗衣机区、客厅北缘、次卧北扩段。

#### 26.1.2 墙面做法

- 原阳台外立面、栏板内侧：**按第 4 节外墙系统**全做（隔汽层 + 岩棉 + 双层石膏板错缝）
- 不得按普通室内隔墙施工
- 与室内隔墙、家政模块交接处：**隔汽层搭接 ≥100mm，专用胶带密封**

#### 26.1.3 地面做法

```text
原阳台楼板基层
  ↓
界面剂
  ↓
防水涂料（满涂，上翻墙根 ≥300mm）
  ↓
闭水试验 ≥24h → 验收 → 拍照
  ↓
1% 找坡向地漏
  ↓
防水保护层
  ↓
地砖 / 地板（按区域）
```

#### 26.1.4 其他要求

| 项目 | 要求 |
|------|------|
| 外窗 | **更换为断桥铝上下分格系统窗**（05 §3）；物业审批后施工；节点见 02 §14–15 |
| 荷载 | 避免大石材、满水浴缸等超重做法 |
| 晾晒 | 整体封闭后无室外晾晒；需配烘干机或另定方案 |
| 物业 | 封闭前须确认审批及外立面统一要求 |

#### 26.1.5 验收看什么

- [ ] 防水闭水通过，已拍照
- [ ] 地面找坡 ≥1%，地漏通畅
- [ ] 北向墙面按外墙标准施工，隔汽层连续
- [ ] 与楼下交接防水翻边到位

### 26.2 家政模块：洗衣区 + 衣帽间

#### 26.2.1 位置关系

- **上**：洗衣机区（封闭阳台内，室内家政间）
- **中**：水平实墙隔断 + 门
- **下**：红框衣帽间（左墙竖向，介于洗衣区与玄关之间）
- **背面**：结构外墙（不另砌堵墙，直接做外墙标准饰面）

#### 26.2.2 洗衣区 ↔ 衣帽间水平隔断（厨卫墙 + 门）

**墙体分类：** 厨卫墙（湿区侧）+ 室内隔墙（干区侧）

**推荐剖面：**

```text
                    ┌─ 洗衣区侧（湿区）─┐
75 轻钢龙骨 或 200mm 加气块
  ↓
湿区：地面防水上翻 ≥300mm；墙面防水（≥1200mm 或整墙到顶）
  ↓
湿区饰面：瓷砖 或 防水腻子 + 耐水漆
  ↓
干区（衣帽间侧）：12mm 耐水石膏板 → 腻子 → 乳胶漆
  ↓
门洞：预留标准门洞；门向洗衣区开启
```

**门的要求：**

| 项目 | 要求 |
|------|------|
| 类型 | 平开防水门（推荐）或推拉门 / 谷仓门（玄关窄时） |
| 材质 | 免漆防潮门，或铝框玻璃门 |
| 门套 | 不锈钢 / 铝合金；底部设挡水条 |
| 门坎 | 洗衣区有地漏时，门坎 20–30mm |
| 开启 | **向洗衣区开**，衣帽间侧保持完整柜墙 |
| 通风 | 洗衣区设排气扇或新风排风口，不依赖常开门 |

**禁止：**

- 仅用柜体隔断（不隔潮、不隔音）
- 未做防水即贴砖或封板
- 门洞开在进水管正下方且无检修措施

#### 26.2.3 衣帽间围合隔墙（室内隔墙 · ④ 类）

**面向玄关 / 走廊的堵墙（新建）：** 采用 **[§3.4.5 衣帽间薄墙方案 B](#345--衣帽间新建围合墙薄墙)**（推荐）。

```text
衣帽间侧 / 走廊侧
  ↓ 12mm 石膏板
  ↓ 18mm 欧松板（整面或柜体/门套/挂衣杆区域）
  ↓ 50 轻钢龙骨 + 40/50mm 岩棉
  ↓ 12mm 石膏板
```

| 指标 | 要求 |
|------|------|
| 总厚度 | **90–100mm**（含饰面） |
| 极限薄 | 75mm（方案 A，仅轻分隔） |
| 贴卧室 | 110–130mm（方案 C，隔音加强） |
| 进门净宽 | 含柜体、门开启 **≥1200mm** |
| 与地面 | 结构胶 + 防潮处理 |

**备选：** 200mm 加气块（更稳、隔潮略好），但占空间；通道窄时优先轻钢龙骨。

**禁止：** 仅柜体隔断、无欧松板固定门套/挂衣杆。

#### 26.2.4 衣帽间靠外墙背面（不另砌堵墙）

直接在结构墙面上做 **第 4 节外墙系统**：

```text
结构墙 → 界面剂 → PE 隔汽膜 → 减振垫 + 75 龙骨
  → 50mm 憎水岩棉 → 双层 12mm 石膏板错缝 → 饰面 → 柜体
```

**柜体安装：**

- 柜体与墙面留 **20–30mm** 透气缝，或柜背贴防潮膜 / 铝箔膜
- 挂衣杆、大柜体：**18mm 欧松板加固**，封板前拍照

#### 27.2.5 施工顺序（家政模块）

```text
1. 封闭阳台地面防水 + 闭水 → 拍照
2. 结构外墙标准施工（洗衣区 + 衣帽间背面）→ 隔汽层验收 → 拍照
3. 水平隔断墙砌筑 / 龙骨安装
4. 湿区墙面防水 → 验收 → 拍照
5. 欧松板加固（洗衣机、柜体、挂衣杆）→ 拍照
6. 穿墙套管 / 水电预留
7. 封板 / 贴砖（湿区）/ 装门
8. 衣帽间柜体安装
```

| 门洞 | 若为大开口，考虑折叠门 / 静音移门补隔声 |

#### 27.3.3 主卫门改向

| 项目 | 内容 |
|------|------|
| 墙体类型 | **厨卫墙** |
| 施工前 | 确认门洞所在墙非承重；排水立管位置不变 |
| 要求 | 新门洞防水、门套防潮；与书房走廊错开避免对门 |
