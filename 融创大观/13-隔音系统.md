# 13 · 隔音系统

> 最后更新：2026-06-28  
> 文档状态：✅ 方案深化（界面清单 + 设备噪声 + 索引；墙体构造见 02 号）

---

## 1. 目标

- 汇总全屋 **空气声** 控制策略：哪面墙/哪个界面用哪套标准
- 明确 **已决策不做** 的项（地面浮筑隔音）
- 与 **FP、两联供、ERV、管道、门窗、吊顶** 等设备噪声交叉对齐
- 封板前可验收、可拍照

**墙体剖面、岩棉、双层石膏板、减振垫做法不在此重复**，见 [02-墙体系统 §4–§7、§9](02-墙体系统.md)。

---

## 2. 适用范围

| 包含 | 不包含 |
|------|--------|
| 外墙、分户墙、电梯/楼梯墙、室内隔墙 **隔声标准选用** | 墙体完整施工剖面（02 号） |
| 设备噪声（FP×5、设备带、ERV、管道） | **地面浮筑/隔音垫**（03 已明确 **不做**） |
| 书房/次卧等 **升级隔墙** | 门窗隔声量化（待 [05 门窗系统](05-门窗系统.md)） |
| 设备减振、管道固定 | 吊顶隔声详图（待 [04 顶面系统](04-顶面系统.md)） |

---

## 3. 设计原则

| 原则 | 说明 |
|------|------|
| **界面分治** | 六类墙体各用各的标准（02 §3），禁止混用 |
| **分户只隔音不保温** | 东 T8/R34、西 **T18** — 不做外墙保温套系 |
| **减振先于吸声** | FP、主机 **底座减振**；龙骨 **减振垫** 隔断刚性声桥 |
| **地面不做隔音** | 不设浮筑楼板、隔音垫（[03 §1](03-地面系统.md)；决策 **2026-06-28-009**） |
| **设备带不传振** | [A]/[B]/erv 与分户墙 **禁止硬连接**（[09 §5.2](09-空气能及水系统.md)） |
| **孔洞必封** | 穿墙套管 + 发泡 + 密封；**禁止背靠背插座**（漏音 + 结露，02 §12） |

## 4. 界面清单 · 按墙体类型

> 模型 `设计中1`；围护认定见 [22 拆改方案](22-拆改方案意向.md) §2.2。

| 墙体类型 | 位置示例 | 隔声做法 | 保温 | 详文 |
|----------|----------|----------|------|------|
| **外墙** | 南/北外轮廓、北阳台原外立面 | 隔汽 + 减振垫 + 75 龙骨 + 50 岩棉 + **双层 12 石膏板 + 阻尼层** | 是 | 02 §4 |
| **分户墙** | **西 T18**、**东 T8/R34** | 75 龙骨 + ≥60 kg/m³ 岩棉 + 减振垫 + **双层石膏板错缝** | **否** | 02 §5 |
| **电梯/楼梯墙** | 靠井道、楼梯间冷墙 | **同外墙** | 是 | 02 §6 |
| **室内普通隔墙** | 卧室之间、通道 | 75 龙骨 + 50 岩棉 + 双面双层石膏板 | 否 | 02 §7、§3.4.3 |
| **室内隔音升级** | **书房扩大墙**（邻公区/走廊） | 75 龙骨 + 50 岩棉 + **双面双层错缝 + 阻尼层** | 否 | 02 §27.3.2 |
| **北阳台半外墙** | 封闭后原外立面、洗衣区背面 | **情形 A → 全按 §4 外墙** | 是 | 02 §3.4.4 |
| **北阳台新建隔断** | 洗衣区↔客厅等 | 100 轻质砖 + 50 岩棉 + 防潮双层板 | 加强 | 02 §3.4.4 情形 B |
| **衣帽间薄墙** | 面向玄关堵墙 | 方案 B/C（要减噪选 **C 隔音加强**） | 否 | 02 §3.4.5 |
| **厨卫墙** | 卫间、厨房湿区 | 防水为主；挂装区 OSB | 否 | 02 §8 |
| **设备墙** | FP 贴墙侧 | OSB + 同上隔墙/分户墙标准 | 视位置 | 02 §9、[06](06-设备墙系统.md) |

## 5. 逐房间 · 隔声关注点

| 房间 | 主要噪声源 | 关键界面 | 要求 |
|------|------------|----------|------|
| **客厅** | FP AC1、邻居（东/西）、室外 | FP **东墙**；分户墙若相邻 | FP **减振垫**；分户墙 §5 |
| **次卧** | FP AC2 贴 **T18**、邻居、北阳台 | **西墙 = 分户墙** | FP 贴 T18 侧：**减振 + 分户墙做法**；**禁止** 当外墙做保温 |
| **主卧** | FP AC4、邻居 | 西墙 FP；东/西分户段 | 同分户墙；上下可调格栅减吹感 |
| **书房** | 公区噪声、FP AC3 | **新建扩大墙** 隔音升级 | 02 §27.3.2 双层 + 岩棉 |
| **厨房** | FP AC5、油烟机（独立） | FP **东墙** | 远离灶台；开放厨房注意 **门/柜体** 补隔声（待 05） |
| **主卫/客卫** | 排水、邻居管道 | 厨卫墙 + 排水管 **隔声棉**（若穿墙） | 02 §8；管道见 §6 |
| **洗衣区** | 洗衣机、排水 | 与衣帽间 **实墙 + 防水门** | 02 §26.2；**禁止柜体隔断** |
| **衣帽间** | 邻玄关走廊 | 薄墙方案 B/C | 贴卧室侧可用方案 C |
| **玄关** | 入户、弱电箱 | T18 分户墙 | 电箱不背靠背 |
| **设备带** | [A] ac_hp、[B] dhw、erv | 北墙；与主卧/次卧分户墙关系 | **减振垫**；禁止硬连接传振至卧室 |

## 6. 设备与管道噪声

> 坐标：[`20260628_MEP设备位置.csv`](../融创大观-3D/模型/全屋/方案推敲/20260628_MEP设备位置.csv)

### 6.1 风机盘管 FP×5

| 项目 | 要求 | 详文 |
|------|------|------|
| 安装 | **落地 LM**，背板贴设备墙 OSB | [07 §3.3](07-风机盘管系统.md) |
| 减振 | **橡胶减震垫**，禁止硬块顶死 | 07 §3.3 |
| 噪声倾向 | FP-68 **40 dB(A)**；FP-51 **37**；FP-34 **36**（国标名义） | 07 §3.2 |
| 贴墙侧 | 次卧 **T18 西墙** 最敏感 — 减振 + 分户墙填充满岩棉 | 06 §6.3 |
| 检修 | 前方净空 **≥ 600 mm** | 07 §3.3 |

### 6.2 设备带主机

| 设备 | 减振 | 隔声备注 |
|------|------|----------|
| [A] 两联供 `ac_hp` | 底座橡胶垫 | 夜间运行；与 [B] 间距 ≥200 mm |
| [B] 生活热水 `dhw` | 同上 | 独立主机，禁止共底座硬连接 |
| ERV `erv` | 同上 | 350 m³/h；风管消声见 08 |
| 分集水器柜 | 柜体不传振至墙 | 泵体振动 primarily 在柜内 |

详见 [09 §5.2](09-空气能及水系统.md)、[08](08-新风系统.md)。

### 6.3 水管与排水

| 类型 | 隔声做法 |
|------|----------|
| 空调水 PPR DN25–32 | 穿墙 **套管 + 柔性接头**；支架加 **橡胶垫** |
| 生活热水 / 软水 | 同左；立管井道内可选 **隔声棉包裹** |
| 冷凝水 PVC | 固定牢，避免空管敲击；汇流处留检修 |
| 洗衣机/厨房排水 | 排水管贴墙固定；洗衣区门 **密封条** |

### 6.4 新风

- 风管穿墙：**套管 + 发泡 + 密封**（02 §13）
- 低位送风口勿直吹床头；**ERV 主机噪声** 在设备带，靠分户墙减振

## 9. 施工要求

```text
1. 按 §4 界面清单确定每面墙类型 → 对照 02 对应章节
2. 龙骨安装前铺减振垫；岩棉 **填满** 空腔
3. 设备定位：FP 减振垫 → 贴墙 → 控制线盒
4. 穿墙：套管 + 发泡 + 密封；穿墙后拍照
5. 封板前：欧松板 + 隔声腔体验收 → 拍照（21 号）
6. 设备调试：FP 夜间档位噪声主观验收（各卧室）
```

与 [15-施工顺序](15-施工顺序及隐蔽验收.md)、[02 §17](02-墙体系统.md) 一致。

---

## 10. 禁止事项

- **地面浮筑隔音**、隔音垫（03 决策）
- 分户墙 **做外墙保温套系**（T18、东墙）
- `75 龙骨 + 单层 9.5 mm 石膏板 + 不填岩棉`（02 §10.4）
- FP/主机 **无减振** 硬顶地面或硬顶墙
- 设备带与分户墙 **钢架硬连接** 传振
- **背靠背插座**（分户墙、外墙）
- 穿墙 **不打套管、不密封**
- 洗衣区 **柜体代替隔断墙**

## 13. 后期检修

- FP 过滤网、接水盘维护时不破坏减振垫
- 穿墙密封若开裂 → 补发泡 + 密封胶（防漏音、防结露）
- 换窗时保持 **窗框发泡 + 密封**（05 号）
