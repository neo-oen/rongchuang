**问题描述：** 客厅 FH1 地暖盘管应采用何种平面走法？多版自动生成的路径存在交叉、无回程、穿管等问题。

**可选方案：**

| 方案 | 优点 | 缺点 | 预算影响 |
|------|------|------|----------|
| A 单根蛇形 + 侧边折返 | 实现简单 | 竖向回程横穿横管，视觉交叉 | 无 |
| B **回字形**（单管矩形螺旋，远端折返回程） | 行业常见；单路闭式回路；符合手画示意 | 中心区域放线需避免交叉 | 无 |
| C 双管平行并排 | 视觉清晰 | **非本项目标准**；与回字混淆 | 无 |

**最终决策：** 采用 **方案 B · 回字形铺管（huizi / rectangular_spiral）** 作为全屋地暖盘管标准。**不使用双管平行并排。**

**修订说明（2026-06-28）：** 初稿误记为「双管平行」，已更正为回字形。

**决策人：** 甲方 / 方案讨论确认
