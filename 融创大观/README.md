> 版本：v0.1（初始化）

> 项目阶段：毛坯新装 · 方案深化 + 施工前准备

---

## 这是什么

本项目不是普通装修记录，而是一套**可持续迭代的工程手册**，用于把融创大观住宅装修过程中的：

- 设计方案
- 施工标准
- 材料选择
- 设备定位
- 隐蔽工程验收
- 拍照留档
- 检修维护要求

全部文档化，形成甲方、施工方、暖通、新风、水电、柜体、门窗等各方都能看懂和执行的统一标准。

**可视化与物料**见并列目录 [`融创大观-3D/`](../融创大观-3D/README.md)（SketchUp 模型、效果图、贴图、物料清单）。

---

## 项目底层目标

```text
长期舒适 + 低能耗 + 低噪音 + 不结露 + 不发霉
+ 可检修 + 可验收 + 可追责 + 少返工 + 后期维护成本低
```

## 如何使用本文档

1. **施工前**：施工方按对应系统文档施工，有疑问先群里确认，不得自行变更。
2. **施工中**：甲方现场代表按 [21-拍照留档规则](./21-拍照留档规则.md) 拍照，按 [19-现场沟通记录](./19-现场沟通记录.md) 记录。
3. **封闭前**：对照各系统「隐蔽验收」章节逐项检查，全部通过才允许封板/封槽/回填。
4. **变更时**：所有不可逆变更写入 [20-问题决策记录](./20-问题决策记录.md)，群里确认后执行。
5. **深化时**：每次只深化一个系统，更新对应文件，同步更新本 README 状态表。
6. **建模时**：在 [融创大观-3D](../融创大观-3D/README.md) 更新对应 SketchUp 场景；物料参数同步 [16-材料品牌型号库](./16-材料品牌型号库.md) 与 `融创大观-3D/物料/清单/`；节点模型归档并更新 [17-节点大样图库](./17-节点大样图库.md)。
