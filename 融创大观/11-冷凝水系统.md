# 11 · 冷凝水系统

> 最后更新：2026-06-28  
> 文档状态：✅ 方案深化（FP 冷凝水路由）

---

## 1. 目标

- 5 台 FP 冷凝水 **独立收集、有组织排放**
- 坡度 ≥ 1%、存水弯、可检修
- 与 [07-风机盘管系统](./07-风机盘管系统.md) 联动

---

## 2. 适用范围

| 包含 | 不包含 |
|------|--------|
| FP×5 冷凝水 | 生活污水（10 号） |
| 本阶段路由与排放点 | 新风、空气能主机冷凝（待设备选型后补） |

---

## 3. 设计原则

```text
坡度 ≥ 1%
独立管路（各 FP 独立支管，可局部汇流）
可检修（存水弯、检修口）
防臭（P 型/S 型存水弯）
禁止倒坡
禁止接入雨水、不明确地漏
禁止封死不可维修
```

**本项目排放策略**（2026-06-28）：

- **上排** 客厅、次卧 → 汇流后 → **客卫地漏**
- **主卧** → **主卫地漏**（独立支管）
- **下排** 书房 → **客卫地漏**
- **厨房** → **厨房下水汇总 DRAIN-HUB-KIT**（与 RO 浓水等同汇流后入水槽排水）

---

## 4. 路由（R-COND-01～05）

> 机器可读：[`mep_pipes.json`](../融创大观-3D/模型/全屋/方案推敲/mep_pipes.json) `condensate_routes`

```text
                    北 ↑
    ┌──────────────────────────────────────┐
    │  AC1客厅    AC2次卧        AC4主卧    │
    │    │         │              │        │
    │    └────┬────┘              │        │
    │         │  COND-TRUNK-U     │        │
    │         ↓ (南墙下行)         ↓        │
    ├─────────┼───────────────────┼────────┤
    │  AC3书房 ──→ 客卫地漏       主卫地漏  │
    │  AC5厨房 ──→ DRAIN-HUB-KIT           │
    └──────────────────────────────────────┘
                    南 ↓
```

| 路由 ID | FP | 管径 | 路径摘要 | 排放点 |
|---------|-----|------|----------|--------|
| R-COND-01 | AC1 客厅 | DN25 PVC | FP接水盘 → 东/南墙下行 → 汇入 COND-TRUNK-U | 客卫地漏 |
| R-COND-02 | AC2 次卧 | DN25 | FP → 西墙 → COND-TRUNK-U | 客卫地漏 |
| R-COND-03 | AC3 书房 | DN20 | FP → 短距离 → 客卫 | 客卫地漏 |
| R-COND-04 | AC4 主卧 | DN25 | FP → 西墙 → 主卫方向 | 主卫地漏 |
| R-COND-05 | AC5 厨房 | DN20 | FP → 厨房下水汇总 | DRAIN-HUB-KIT |

### 4.1 汇流点

| ID | 位置 | 作用 |
|----|------|------|
| **COND-TRUNK-U** | 上排南缘 y≈130 SU（约 3250 mm） | AC1+AC2 冷凝水干管，西向至客卫 |
| **DRAIN-HUB-KIT** | 厨房水槽下 mm (3050,1850) | AC5 冷凝水 + RO浓水 + 软水再生 |
| **客卫地漏** | mm (5550,1850) | AC1/2/3 最终排放 |
| **主卫地漏** | mm (11175,1850) | AC4 最终排放 |

### 4.2 施工要求

| 项目 | 要求 |
|------|------|
| 坡度 | **≥ 1%**，全程无倒坡 |
| 存水弯 | 每台 FP 接水盘下或汇流前设 P 型弯 |
| 管径 | DN20（单台短距）/ DN25（汇流干管） |
| 固定 | 横管支架间距 ≤ 0.5 m |
| 保温 | 穿越非空调区、易结露段可选 9 mm 橡塑 |
| 试水 | 灌水试验，接口无渗漏 |

---

## 5. 材料

| 材料 | 规格 |
|------|------|
| 冷凝水管 | PVC-U，DN20 / DN25 |
| 存水弯 | PVC 同径 |
| 三通/弯头 | 同材质 |
| 专用冷凝水提升泵 | **不采用**（全程重力，保证坡度） |

---

## 6. 禁止事项

- 倒坡、无存水弯
- 接入雨水管、马桶污水管（除地漏污水区）
- 多台 FP 共用一个无检修存水弯
- 未做坡度试验即封吊顶/墙

---

## 7. 隐蔽验收

- [ ] 5 路 FP 冷凝支管走向

- [ ] 存水弯位置
- [ ] 客卫/主卫/厨房汇总点接口
- [ ] 灌水试验记录
