### 4.2 暖通一体温控（2026-06-28 · FP + 地暖 + 新风）

> 详见 [07-风机盘管系统 §7](./07-风机盘管系统.md)、[08-新风系统 §8](./08-新风系统.md)、[`hvac_control.json`](../融创大观-3D/模型/全屋/方案推敲/hvac_control.json)

| 项目 | 要求 |
|------|------|
| 一体温控器 | 5 块（TC-Z1–Z5），86 型，**FP + 地暖 + 新风风阀** 三合一 |
| 安装高度 | 完成面 **1.4 m** |
| 底盒加固 | 18 mm 欧松板（02 号 §11） |
| FP 控制线 | 每区 ≥ 3 芯，TC → FP，独立穿管 |
| **新风风阀线** | 每区 ≥ 2 芯，TC → **MD-Z1～Z5**，独立穿管 |
| 执行器信号线 | 弱电星型：设备带端子排 ↔ 各 TC |
| ERV 集控 | 设备带 **新风集控模块** ↔ erv（RS485 或 0–10 V） |
| 强电弱电间距 | ≥ 300 mm；交叉垂直 |

**禁止**：温控器与 FP 同墙；厨房 TC 装灶后；强电弱电同管。

### 4.3 FP 专用回路（2026-06-28 · 通用 GB/T 19232）

> 详见 [07-风机盘管系统 §6](./07-风机盘管系统.md)、[`物料清单_风机盘管与温控系统.md`](../融创大观-3D/物料/清单/物料清单_风机盘管与温控系统.md)

| 回路 | FP | 房间 | 电机 W | 线径 | 保护 | 备注 |
|------|-----|------|--------|------|------|------|
| L-FP1 | FP-68 | 客厅 | 60 | 1.5 mm² | 10 A 漏电 | 专用回路 |
| L-FP2 | FP-51 | 次卧 | 51 | 1.5 mm² | 10 A 漏电 | 专用回路 |
| L-FP3 | FP-34 | 书房 | 39 | 1.5 mm² | 10 A 漏电 | 专用回路 |
| L-FP4 | FP-51 | 主卧 | 51 | 1.5 mm² | 10 A 漏电 | 专用回路 |
| L-FP5 | FP-34 | 厨房 | 39 | 1.5 mm² | 10 A 漏电 | 远离灶台，留检修空间 |

**底盒**：FP 侧预留控制线接线盒；与 TC 控制线 **分管敷设**，强电弱电间距 ≥ 300 mm。

### 4.7 踢脚灯带（2026-06-28 · 嵌入式暗装）

> 构造见 [02-墙体系统 §10.6](./02-墙体系统.md)

| 项目 | 要求 |
|------|------|
| 适用 | **干区**嵌入式踢脚灯带；**卫间不做** |
| 灯带 | DC **24 V** LED；预留 **16 线管** 至驱动 |
| 驱动 | 专用回路或汇总至 **弱电/照明回路**；**≤ 60 W/路** 倾向 |
| 控制 | 可与全屋智能 / 场景联动；与 TC **独立** |
| 线管 | 封板前预埋；驱动位置 **可检修** |

### 4.8 玄关 T18 · 强弱电箱（2026-06-28 · 上强下弱）

> 详 [06-设备墙系统 §5.2](./06-设备墙系统.md)

| 项目 | 要求 |
|------|------|
| 位置 | 入户 **玄关**，**西侧分户墙 T18** 内侧 |
| **强电箱** | 底边 **1600 mm**；回路 **≥ 20 位**（含 §4.3–4.6 + 客厅 **网络区电源**） |
| **弱电箱** | 底边 **400 mm**；**300×250 mm+**；**仅光猫 + 光纤终端** |
| 竖向净距 | 强电底 − 弱电顶 **≥ 500 mm** |
| 弱电箱电源 | **220 V / 10 A**，**仅供光猫** |
| 入户 | 市电总开、光纤、可视对讲（若有）同区 |
| 基层 | T18：**贴层暗装** 或 **明装金属箱**（06 §5.2） |

### 4.9 网络拓扑 · 电视墙汇聚（2026-06-28）

> 详 [06-设备墙系统 §5.3](./06-设备墙系统.md)

| 项目 | 要求 |
|------|------|
| 汇聚点 | **客厅电视墙区**：路由器 + **千兆交换机** + 配线/理线 |
| 弱电箱 | **仅光猫**；不放路由、交换机、AP 主机 |
| 上联 | 弱电箱 → 电视墙 **≥ 2 根六类**（1 用 1 备），**20 线管**，宜 **单段直达** |
| 房间线 | 各信息点 **1 管 1 线** **直达电视墙**；**禁止** 汇至弱电箱再跳线 |
| 电源 | 电视墙网络区 **独立五孔**（§5.1）；与光猫电源 **分开** |
| 机柜 | 矮柜/开放格；**通风**；设备可重启、可换线 |
| 与暖通 | TC / FP 控制为 **硬线**，不走 IP；与网络 **独立** |

**信息点倾向**：客厅 2 · 次卧 1 · 主卧 1 · 书房 2 · 厨房 1（可选）。

**强电回路汇总（专用）**：L-FP1～5 · L-AC-HP · L-AC-PUMP · L-AC-AUX（可选）· L-ERV · L-HVAC-CTL · 水柜 · [B] dhw · 小厨宝 · RO/管线机 · 厨电 · 洗衣 · **弱电箱电源** · **客厅网络区电源**。
