# 01 · 总体设计原则

> 最后更新：2026-06-27  
> 文档状态：v0.1

---

## 1. 目标

本文档定义融创大观住宅装修**全屋系统级设计原则**。所有子系统文档（墙体、地面、顶面、机电等）必须与本原则一致；如有冲突，以本原则为第一性原理基准，重大冲突须甲方群里确认后调整。

---

## 2. 适用范围

适用于本项目全部装修范围，包括但不限于：

- 围护系统（墙、地、顶、门窗）
- 机电系统（空气能、风机盘管、新风、冷热水、冷凝水、电气弱电）
- 隐蔽工程、拍照留档、后期检修

---

## 3. 第一性原理

### 3.1 本项目要解决什么

```text
不是「装得好看」
而是「长期住得舒服、能耗低、噪音小、不结露、不发霉、好修、好查、少扯皮」
```

### 3.2 每个方案必须回答的 10 个问题

在采用任何做法之前，必须逐项判断：

| # | 问题 | 不通过则 |
|---|------|----------|
| 1 | 是否影响结构安全？ | 必须专业确认，禁止自行施工 |
| 2 | 是否会造成冷桥、结露、发霉？ | 必须调整方案或加强节点 |
| 3 | 是否会造成噪音传播？ | 必须加强隔音或改变做法 |
| 4 | 是否影响空气能、新风、风机盘管、水管、冷凝水？ | 必须联动各系统文档确认 |
| 5 | 是否方便后期检修？ | 必须预留检修口 / 阀门 / 可拆卸节点 |
| 6 | 是否可以在封闭前验收？ | 必须调整施工顺序或留验收窗口 |
| 7 | 是否有照片留档？ | 封闭前必须拍照，否则不得封闭 |
| 8 | 是否会让施工方后期扯皮？ | 必须群里文字确认 |
| 9 | 是否增加预算？增加是否值得？ | 必须甲方确认 |
| 10 | 是否有更简单、更稳妥的方案？ | 优先采用更简单稳妥方案 |

## 4. 系统设计原则

### 4.1 围护系统原则

| 原则 | 要求 |
|------|------|
| 保温 + 隔音 + 防结露一体考虑 | 外墙不能只贴保温，必须同时处理隔汽层、冷桥、露点位置 |
| 六类墙体分类施工 | 外墙、分户墙、电梯/楼梯墙、室内隔墙、厨卫墙、设备墙——各自标准不同，禁止混用 |
| 禁止 XPS 作为内保温主体 | XPS 仅可在特定节点（如飘窗顶部）局部使用，不得替代外墙岩棉体系 |
| 隔汽层必须连续 | PE 隔汽膜搭接 ≥ 100mm，穿孔必须补强密封 |
| 双层石膏板错缝 | 禁止单层 9.5mm 石膏板替代外墙封板 |

### 4.2 机电系统原则

| 原则 | 要求 |
|------|------|
| 设备集中布置 | **[A] 两联供** 与 **[B] 空气能热水器** 两台独立主机，并列于厨房后北侧设备带；缓冲水箱（[A]）与储热水箱（[B]）**不得混用** |
| 各房间立式明装风机盘管 | 位置、加固、水管、冷凝水、电源、控制器必须提前定位 |
| 冷凝水独立排放 | 坡度 ≥ 1%，独立管路，可检修，有存水弯，防臭 |
| 管线走向提前规划 | 与吊顶、梁、柜体、检修口联动，禁止临时走管 |
| 所有设备考虑检修 | 平台、吊顶、设备墙必须预留检修空间和检修口 |

### 4.3 新风系统原则

| 原则 | 要求 |
|------|------|
| 全热交换 | 倾向采用全热交换新风，具体型号待确认 |
| 低位送风、高位排风 | 送排风口位置与家具、柜体、吊顶联动 |
| 管路可检修 | 禁止封死在不可拆卸结构内 |
| 穿墙节点标准做法 | 套管 + 发泡 + 密封胶，见 [02-墙体系统](./02-墙体系统.md) |

### 4.4 防潮防结露原则

```text
室内 → 隔汽层 → 保温/岩棉层 → 结构墙 → 室外
```

重点位置：外墙、飘窗、窗框、风口、北向房间、柜体后部、设备背后、新风管周边、冷凝水管周边。

目标：防止发霉、长毛、墙纸鼓包、乳胶漆起皮、柜体背后潮湿、冬季窗边结露。

## 5. 施工管理原则

### 5.1 确认机制

```text
口头沟通 ≠ 施工依据
群里文字确认 = 施工依据
```

以下必须群里确认后方可施工（详见 [00-项目总览](./00-项目总览.md) 第 5 节）：

拆改、水电、新风、冷凝水、空气能、风机盘管、防水、吊顶、保温隔音、检修口、预算增加、设备位置调整、管线走向调整、墙体开洞、柜体加固、门窗更换、所有不可逆施工。

### 5.2 封闭前原则

```text
未验收 → 禁止封闭
未拍照 → 禁止封闭
未确认 → 禁止封闭
```

### 5.3 材料替代原则

- 任何材料替代必须书面说明替代条件和差异
- 禁止用普通吸音棉、普通玻璃棉、低密度岩棉替代憎水岩棉
- 禁止用过薄或非镀锌龙骨替代标准轻钢龙骨
- 禁止用单层薄石膏板替代双层错缝石膏板（外墙系统）
- 替代方案必须甲方确认

### 5.4 深化节奏

1. 不追求一步到位，先骨架后深化
2. 每次只深化一个系统，避免混乱
3. 所有现场口头沟通转成文字记录（[19-现场沟通记录](./19-现场沟通记录.md)）
4. 所有问题决策写入 [20-问题决策记录](./20-问题决策记录.md)

## 6. 协作分工原则

| 角色 | 原则 |
|------|------|
| 甲方总负责人 | 系统取舍、预算优先级、重大变更拍板、关键节点验收 |
| 甲方现场代表 | 巡查、拍照、日常沟通、问题上报——**不上报等于未发生** |
| 乙方施工方 | 按确认方案施工、提出可行性反馈、报价——**不得自行变更方案** |

## 7. 禁止事项（全屋通用）

| # | 禁止事项 | 原因 |
|---|----------|------|
| 1 | 未经确认擅自拆改、开洞 | 结构安全、物业规则 |
| 2 | 外墙不做隔汽层或隔汽层不连续 | 结露、发霉 |
| 3 | 冷凝水倒坡或接入不明确排水点 | 漏水、返臭 |
| 4 | 背靠背插座（尤其外墙） | 漏风、漏音、冷桥、结露 |
| 5 | 直接打洞穿管不用套管 | 破坏隔汽层、气密性差 |
| 6 | 隐蔽工程封闭前不拍照 | 无法追责、无法检修 |
| 7 | 设备不做基层加固直接挂装 | 脱落风险 |
| 8 | 检修口、阀门封死在不可拆卸结构内 | 后期无法维护 |
| 9 | 用口头沟通代替群里确认 | 扯皮、返工 |
| 10 | 未经验收强行封闭 | 隐患不可查 |
