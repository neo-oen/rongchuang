# 09 · 空气能及水系统

> 最后更新：2026-06-28  
> 文档状态：✅ 方案+通用深化（**仅 [A] 冷暖两联供**；生活热水见 10 号 **[B] 空气能热水器**）

---

## 0. 与 [B] 空气能热水器的区别

| | **[A] 本文件** | **[B] 10 号文档** |
|---|----------------|-------------------|
| 设备 | **空气能两联供** | **空气能热水器**（专用热水机） |
| 数量 | **第 1 台**空气能 | **第 2 台**空气能（独立采购、独立安装） |
| 水箱 | 100 L **缓冲**（二次侧，非饮用） | 150–200 L **储热水箱** |
| 末端 | FP×5 + 地暖 FH×5 | 龙头、淋浴、洗碗机 |
| 管路 | 上排 y=148/144 | 南排 y=48/50 |
| 模型 | `ac_hp` | `dhw` |

**禁止**：用一台机器同时承担 [A]+[B]；共用一次侧/二次侧/生活热水管路。

---

## 1. 目标

- **[A] 冷暖两联供**：主机 + 缓冲耦合 → 二次侧 **并联** 供应 FP×5 + 地暖 FH1–FH5
- 与 **[B] 生活热水** 完全分离（独立主机、独立管路）
- 设备平台布置、二次侧路由、阀门、试压、电气、验收

---

## 2. 适用范围

| 包含 | 不包含 |
|------|--------|
| 厨房后北侧设备平台 `[A]` 主机区 | [B] 生活热水 dhw（10 号） |
| 二次侧干管、分集水器接口 | FP 支管细节（07 号） |
| 缓冲水箱、循环泵、过滤器 | 地暖盘管与地面辅材（03 号） |
| 主机化霜排水原则 | 具体品牌（待比价） |

**设计依据**：安徽省 **DB34/T 3949-2021**《空气源热泵供暖空调工程技术规程》；亳州属夏热冬冷向寒冷过渡区，名义制热工况 **-2℃/-3℃**。

---

## 3. 系统架构

```text
室外机（压缩机+风侧）
    ↕ 一次侧
室内耦合单元（缓冲水箱 + 二次循环泵）
    ├─ R-AC-01/02 → 5×FP 并联（AC1–AC5）
    └─ R-AC-03    → 分集水器 FH1–FH5 并联
```

- FP 与地暖 **禁止串联**；与生活热水 **[B] 禁止混接**
- 拓扑：[`mep_topology.json`](../融创大观-3D/模型/全屋/方案推敲/mep_topology.json)

## 4. 负荷估算与通用选型

### 4.3 主机通用选型（[A]）

| 设备 | 通用规格 | 数量 | 位置 |
|------|----------|------|------|
| **冷暖两联供主机** | 名义 **制冷 ≥16 kW**、**制热 ≥18 kW**（-2℃/-3℃ 名义工况）；**A 型或 B 型**一体式；带 **低温启动 + 化霜** | 1 | 厨房后平台 |
| **缓冲耦合水箱** | **100 L**（或与主机一体）；材质不锈钢/搪瓷；带 **补水口、排气、排污、温度传感器接口** | 1 | 与主机同区 |
| **二次循环泵** | 扬程 **≥8 m**，流量 **≥2.5 m³/h**；优先 **屏蔽泵**；常含于主机套餐 | 1 | 主机/水箱内或旁 |
| **Y 型过滤器** | DN32，304 不锈钢滤网 | 1 | 二次侧回水 |
| **自动排气阀** | DN15 或分集水器自带 | 1 | 干管高点 / 分集水器 |
| **定压补水装置** | 闭式系统膨胀罐 + 补水阀（或主机自带） | 1 | 设备带 |
| **电辅热（可选）** | 主机标称制热量 **30%** 管道式辅助（**-7℃ 以下极端日**）；非必须，下单前按厂家方案 | 0–1 | 缓冲水箱内 |

**选型逻辑**：制冷按 FP 同时系数 **10.7 kW** 留 **≥25% 裕量 → 16 kW 级**；制热按地暖 **4.6 kW** + FP 快速补热裕量，**18 kW 级** 名义制热可覆盖；夏冬季不会同时峰值。

## 6. 二次侧水管路由

> 详 [`MEP水管路由与预留点位.md`](../融创大观-3D/模型/全屋/方案推敲/MEP水管路由与预留点位.md) §5 · [`mep_pipes.json`](../融创大观-3D/模型/全屋/方案推敲/mep_pipes.json)

### 6.1 干管

| 路由 | 路径 |
|------|------|
| **R-AC-01** 供水 | `ac_hp.sec_supply`（北口）→ 上排 **y=148**（Z≈2700 mm）→ 5×FP 立管 |
| **R-AC-02** 回水 | 5×FP → 上排 **y=144** → `ac_hp.sec_return`（北口） |
| **R-AC-03** 至分集水器 | [A] 东侧 → `fh_manifold.main_supply` / `main_return`（**并联**，非串联） |
| **R-FH-01** 地暖 | 分集水器 → FH1–FH5 干管 → 回字盘管（`floor_heating.json`） |

**分支规则**：FP 中心 x &lt; 250 SU → **西墙**接入；x ≥ 250 → **东墙**。厨房 AC5 自北墙接西向干管。

### 6.2 管材

| 段 | 材料 | 管径 | 保温 |
|----|------|------|------|
| 二次侧干管 | PPR S3.2 或薄壁不锈钢 | **DN32** | 橡塑 **≥20 mm** |
| FP / 分集水器支管 | 同左 | **DN25** | 橡塑 **≥20 mm** |
| 地暖支管 | PE-RT 阻氧 | **Ø16 mm** | 地面绝热层 |

### 6.3 阀门清单

|----|------|------|------|
| **V-AC-M01** | 球阀 | DN32 | 二次侧总供水 |
| **V-AC-M02** | 球阀 | DN32 | 二次侧总回水 |
| **V-AC-F01～F05** | 球阀×2 | DN25 | 各 FP 供 + 回支路 |
| — | Y 型过滤器 | DN32 | 二次侧回水（V-AC-M02 前） |
| — | 自动排气阀 | DN15 | 干管高点 / 分集水器 |
| FH 支路 | 球阀 + 电热执行器 | 按分集水器 | 03/07 号 |

## 8. 施工顺序

```text
1. 设备平台地面 + 承重确认 + 防水（若有敞露）
2. [A] 主机/缓冲水箱就位 + 减振 + 室外机（若分体）
3. 二次侧 DN32 干管 + 保温 + 过滤器 + 阀门
4. FP 支管 DN25 + 分集水器 R-AC-03 接口
5. 地暖 FH 干管 + 分集水器 5 路（与 03 号同步）
6. 闭式系统补水 + 排气
7. 二次侧试压 0.6 MPa → 冲洗 → 保温外护
8. 强电 + 传感器线 → 联合调试（FP + 地暖 + TC）
```

---

## 9. 试压与调试

| 项目 | 要求 |
|------|------|
| 二次侧试压 | **0.6 MPa**，保压 **30 min**，压降 ≤ **0.05 MPa** |
| 水质 | 闭式系统 **软化水或过滤自来水**；禁止未冲洗直接通地暖 |
| 地暖各路 | 分集水器逐路通水排气；见 03 号 |
| 联合调试 | 制冷：FP 逐台 + 地暖互锁关；制热：地暖主 + FP 补热；记录供回水温差 |

## 11. 禁止事项

- [A] 与 [B] 管路、水箱混接
- FP 与地暖 **串联**
- FH1 干管穿越客卫
- 二次侧裸管（无保温）穿越非设备区
- 未试压、未排气即开地暖/FP
- 过滤器、缓冲水箱排污口封死不可检修

## 14. 后期检修

- 过滤器：**每采暖季前** 清洗一次
- 缓冲水箱：每年检查排污、水质
- 主机：厂家规定保养周期；化霜、冷媒由 **持证暖通** 操作
- 分集水器：各支路阀门可单独关断，便于单路地暖检修
