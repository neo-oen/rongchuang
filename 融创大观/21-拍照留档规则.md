# 21 · 拍照留档规则

> 最后更新：2026-06-27  
> 文档状态：v0.1

---

## 1. 目标

建立本项目**隐蔽工程拍照留档标准**，确保：

- 所有封闭前工程有影像证据
- 后期检修可定位管线、加固点、设备位置
- 出现纠纷时可追溯施工状态
- 甲方、乙方对「封闭前是否合规」有统一判断依据

**核心原则：未拍照 = 未施工完成 = 禁止封闭。**

---

## 2. 适用范围

适用于本项目所有在封闭后无法或难以检查的工程，包括但不限于：

- 墙体保温隔音封板前
- 地面找平 / 地暖 / 管线铺设后、覆盖前
- 吊顶封板前
- 水电管线封槽前
- 新风 / 空调水管 / 冷凝水管封吊顶或封墙前
- 防水施工后、贴砖前
- 设备墙欧松板封板前
- 穿墙套管节点
- 平台设备基础及管线连接

---

## 3. 拍照基本要求

### 3.1 拍摄人

- **主要执行人**：甲方现场代表（妻子）
- **备份**：甲方可随时抽查；乙方可配合补拍但不可替代甲方留档

### 3.2 拍摄时间

- 必须在**封闭前、验收通过后**立即拍摄
- 禁止先封闭后补拍
- 禁止用施工过程中的随手拍代替正式留档

### 3.3 照片质量要求

| 要求 | 标准 |
|------|------|
| 清晰度 | 能看清材料、管线走向、节点做法、标识 |
| 尺寸参照 | 关键位置**必须**有卷尺或标注尺寸（欧松板加固、设备定位、管线间距等） |
| 范围 | 全景 + 局部细节，不能只拍一个角度 |
| 标识 | 注明房间、墙面编号/方位、拍摄日期 |
| 数量 | 每个封闭节点不少于 3 张（全景 1 + 细节 2） |

### 3.4 存储与命名

建议命名格式：

```text
YYYYMMDD_房间_系统_节点_序号.jpg
```

示例：

```text
20260627_主卧_外墙_隔汽层搭接_01.jpg
20260627_客厅_设备墙_欧松板加固_02.jpg
20260627_厨房_冷凝水_坡度测试_01.jpg
```

存储位置：待建立（建议云盘 + 本地双备份，按房间 / 系统分文件夹）

## 4. 分系统拍照清单

### 4.1 墙体系统（封板前必拍）

| # | 拍摄内容 | 必含要素 | 对应文档 |
|---|----------|----------|----------|
| 1 | 隔汽膜整体铺设 | 搭接宽度、胶带密封、是否连续 | [02-墙体系统](./02-墙体系统.md) |
| 2 | 减振垫 + 龙骨安装 | 龙骨间距、减振垫覆盖 | 同上 |
| 3 | 岩棉填充 | 厚度、填充密实度、憎水标识 | 同上 |
| 4 | 欧松板加固位置 | **卷尺标注尺寸**、固定点范围 | 同上 |
| 5 | 插座盒节点 | 发泡胶、密封胶、是否背靠背 | 同上 |
| 6 | 穿墙套管节点 | 套管、发泡、内外密封 | 同上 |
| 7 | 双层石膏板错缝 | 板缝错开位置 | 同上 |
| 8 | 飘窗特殊节点 | 顶部保温、两侧做法、窗台底部 | 同上 |

### 4.2 地面系统（覆盖前必拍）

| # | 拍摄内容 | 必含要素 |
|---|----------|----------|
| 1 | 地面管线走向 | 水管、电管、冷凝水（如有） |
| 2 | 保温层 / 隔音层 | 材料、厚度、搭接 |
| 3 | 地暖铺设（如做） | 间距、回路、边界保温 |
| 4 | 找平前标高 | 卷尺标注完成面标高 |
| 5 | 门槛石基层 | 与相邻房间标高差 |

### 4.3 顶面系统（封板前必拍）

| # | 拍摄内容 | 必含要素 |
|---|----------|----------|
| 1 | 吊顶内管线走向 | 新风管、空调水管、冷凝水管、电管 |
| 2 | 检修口预留位置 | 尺寸、对应设备 |
| 3 | 新风管保温 | 保温材质、厚度、接口 |
| 4 | 吊杆 / 龙骨 | 间距、设备避让 |
| 5 | 梁位处理 | 管线翻梁做法 |

### 4.4 水电系统（封槽 / 封板前必拍）

| # | 拍摄内容 | 必含要素 |
|---|----------|----------|
| 1 | 强电管线走向 | 线管、底盒位置 |
| 2 | 弱电管线走向 | 网线、线管、底盒 |
| 3 | 给水管 / 排水管 | 管径、坡度、固定 |
| 4 | 热水回水管 | 走向、保温（**本项目无回水 → N/A，跳过**） |
| 5 | 阀门位置 | 对应功能标注 |

### 4.5 机电系统（封闭前必拍）

| # | 拍摄内容 | 必含要素 |
|---|----------|----------|
| 1 | 风机盘管安装 | 位置、水管接口、冷凝水管、电源、加固 |
| 2 | 新风主机及管路 | 主机位置、管径、保温、穿墙节点 |
| 3 | 空气能主机及水箱 | 平台布置、管线连接、基础 |
| 4 | 冷凝水全程 | **坡度测试照片**、存水弯、排放点 |
| 5 | 空调水管 | 保温、支架、接口 |

### 4.6 防水系统（贴砖前必拍）

| # | 拍摄内容 | 必含要素 |
|---|----------|----------|
| 1 | 防水涂刷范围 | 高度、翻边、地漏周边 |
| 2 | 闭水试验 | 水位、时间、结果 |
| 3 | 管道周边防水 | 加强层 |

### 4.7 门窗系统

| # | 拍摄内容 | 必含要素 |
|---|----------|----------|
| 1 | 窗框与墙体间隙 | 发泡、密封胶 |
| 2 | 窗台保温处理 | 底部保温做法 |
| 3 | 更换窗口后节点 | 如发生换窗 |

## 5. 关键节点专项要求

### 5.1 欧松板加固（强制）

```text
必须拍：加固板位置 + 卷尺标注 + 对应设备名称
禁止：封板后凭记忆找位置
```

### 5.2 冷凝水（强制）

```text
必须拍：全程坡度 + 存水弯 + 排放点 + 试水结果
禁止：未试水、未拍坡度就封吊顶
```

### 5.3 隔汽层（强制）

```text
必须拍：整体连续性 + 搭接 + 穿孔补强
禁止：隔汽层破损未补就封板
```

### 5.4 穿墙节点（强制）

```text
必须拍：套管 + 发泡 + 内外密封（封饰面前）
禁止：直接打洞不套管
```

## 6. 拍照与验收的关系

```text
施工完成
    ↓
乙方自检
    ↓
甲方现场代表验收（对照各系统隐蔽验收清单）
    ↓
验收通过 → 立即拍照留档
    ↓
甲方确认照片合格
    ↓
允许封闭
```

**验收不通过：整改 → 重新验收 → 重新拍照。**

---

## 7. 禁止事项

| # | 禁止 | 后果 |
|---|------|------|
| 1 | 先封闭后补拍 | 视为未留档，打开复查 |
| 2 | 照片无卷尺 / 无尺寸（加固点） | 照片作废，重拍 |
| 3 | 只拍一张敷衍 | 不满足最低数量要求 |
| 4 | 照片无法辨认房间和节点 | 作废，重拍 |
| 5 | 施工方代替甲方留档后不告知 | 甲方必须亲自确认收到 |
| 6 | 口头说「拍过了」但无照片 | 视为未拍 |

---

## 8. 后期使用

拍照留档用于：

1. **检修定位**：找欧松板、阀门、管线、设备接口
2. **纠纷追溯**：施工做法是否符合确认方案
3. **二次改造**：了解墙内、地面、吊顶内原有状态
4. **经验迭代**：更新本工程手册各系统文档
