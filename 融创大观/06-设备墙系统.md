# 06 · 设备墙系统

> 最后更新：2026-06-28  
> 文档状态：✅ 方案深化（点位清单 + 构造索引；剖面做法见 02 号 §9/§11）

---

## 1. 目标

- 汇总全屋 **挂装 / 贴墙 / 底盒** 设备点位，明确 **18 mm 欧松板** 加固范围
- 区分 **设备墙**（挂装基层）与 **设备带**（厨房后北墙落地平台，见 §4）
- 与 FP、水柜、温控、强弱电、家政模块 **交叉对齐**，封板前可验收、可拍照

**构造标准（隔汽、龙骨、欧松板工艺）不在此重复**，见 [02-墙体系统 §9、§11](./02-墙体系统.md)。

---

## 2. 适用范围

| 类型 | 包含 | 不包含 |
|------|------|--------|
| **设备墙** | FP 贴墙侧、TC 底盒墙、厨房水柜/吊柜墙、卫间挂装、洗衣区、衣帽间、玄关 T18 电箱墙、南向窗帘盒局部 | 设备带落地主机（§4） |
| **非挂装** | 客厅 **移动电视架** + **网络汇聚**（§5.1、§5.3） | 地暖盘管、吊顶新风管（03/04/08 号） |

## 5. 已确认专项

### 5.1 客厅电视 · 移动电视架 + 网络汇聚（2026-06-28 确认）

| 项目 | 决策 |
|------|------|
| 形式 | **落地移动电视架**，**不壁挂** |
| 欧松板 | **不需要** 电视挂装加固 |
| **网络汇聚** | **路由器、交换机、配线架** 实装于 **客厅电视墙区**（§5.3）；弱电箱 **仅光猫** |
| 电位 | 电视区 + 网络矮柜/机柜区：<br>• **2×五孔 16 A**（电视 1 + 网络设备 1，宜分开）<br>• 完成面 **350–400 mm** 墙脚 **或** 矮柜内插座 |
| 网线 | 电视区面板接 **本地交换机**；全屋六类 **汇此**，不回弱电箱 |
| 走线 | 线管沿墙脚/踢脚；HDMI 随移动架理线，不做承重墙大槽 |
| 仍须 OSB | 南向窗帘盒/电机；矮柜或机柜 **挂墙** 时局部 OSB |

> 决策记录：**2026-06-28-010 / 011**

### 5.2 玄关 · 分户墙 T18 · 强弱电箱（2026-06-28 确认）

**位置**：入户玄关，**西侧分户承重墙 T18** 内侧。**上强下弱**；弱电箱 **仅终结光纤 + 光猫**。

| 箱体 | 安装高度（底边距完成面） | 规格倾向 | 备注 |
|------|--------------------------|----------|------|
| **强电箱** | **1600 mm** | 回路 **≥ 20 位** | 含 **客厅网络区电源**（§5.1） |
| **弱电箱** | **400 mm** | **300×250 mm+** | **仅光猫**；**不放** 路由/交换机 |
| 竖向净距 | 强电底 − 弱电顶 **≥ 500 mm** | — | 强弱 **≥ 300 mm** |
| 弱电箱电源 | **220 V / 10 A** | — | **仅供光猫** |

**T18 施工**：承重墙不大开槽；轻钢贴层暗装或明装金属箱（同前）。

**入户关联**：光纤 → 弱电箱光猫 → **≥ 2 根六类**（1 用 1 备）**直管至客厅电视墙**（§5.3）；**V01**、可视对讲与强电同区。

> 决策记录：**2026-06-28-010 / 011**；详 [12 §4.8–4.9](./12-电气弱电系统.md)

### 5.3 弱电拓扑 · 电视墙汇聚（2026-06-28 确认）

**可行，已采用。** 有源设备在客厅；各房间网线 **星型直达电视墙**，不经弱电箱转接。

```text
光纤 → 玄关弱电箱（光猫）
           ↓ 六类 ×1（+1 备用）
客厅电视墙 ─ 路由器 → 交换机 → 配线/理线
           ↑
     次卧/主卧/书房/厨房… 各信息点（单根直达）
```

| 项目 | 要求 |
|------|------|
| 汇聚点 | 客厅 **电视墙区**（与移动电视同区） |
| 上联 | 弱电箱 → 电视墙 **2 根六类同管**（20 线管） |
| 房间线 | **1 管 1 线** 直达交换机侧；**禁止** 先汇弱电箱 |
| 机柜 | 矮柜/开放格/墙挂小机柜；**通风**、可断电重启 |
| 线材 | 六类非屏蔽；单段 **≤ 90 m** |

**信息点倾向**：客厅 2 · 次卧 1 · 主卧 1 · 书房 2 · 厨房 1（可选）→ 全部 **→ 电视墙**。

**禁止**：路由/交换放弱电箱；房间线汇弱电箱再跳线；光猫—路由 **多段面板跳线**（宜箱内直插或单段直达）。

## 9. 施工顺序

```text
1. 设备定位会签（FP、TC、水柜、电箱、移动电视架脚位）
2. 轻钢龙骨 / 砌体完成后 → 按 §6 清单安装欧松板 → 拍照（卷尺+设备名）
3. 强电：玄关 T18 箱位 + FP 专用回路 + 设备带电源
4. 弱电：玄关弱电箱（光猫）+ **弱电箱→电视墙上联管** + 各房间→电视墙六类 + TC×5 控制线管
5. 生活水 / 空调水 / 冷凝水管 → 试压（各系统文档）
6. FP 落地安装 + 减振
7. 封石膏板 / 厨卫贴砖 → 隐蔽验收
```

与 [15-施工顺序及隐蔽验收](./15-施工顺序及隐蔽验收.md) 总序一致；**设备墙 OSB 必须在封板前完成**。

## 10. 禁止事项

- FP、浴室柜、吊柜、电箱 **无 OSB 即封板**
- TC 与 FP **同墙**
- 强弱电 **同管** 或间距 **< 300 mm** 平行敷设
- 电视 **壁挂**（已改为移动架）
- 在 **T18 承重墙** 未经确认 **大开槽**
- 设备带主机 **硬连接** 传振至卧室分户墙
- 路由器、交换机放 **弱电箱**
- 各房间网线 **先汇弱电箱再跳线**至客厅
- 光猫与路由器之间 **多段面板跳线**（宜单段或箱内直插）
- 弱电箱 **无 220 V 电源**（光猫无法工作）
