# 14 · 防潮防结露系统

> 最后更新：2026-06-28  
> 文档状态：✅ 方案深化（设计参数 + 风险矩阵 + 机电保温汇总；节点构造见 02 号）

---

## 1. 目标

- 防止 **气态结露**（冬季墙内/窗边发霉）与 **液态水**（渗漏、积水）混淆处理
- 明确亳州 **设计温湿度**，指导隔汽层、保温、冷桥切断
- 汇总 **外墙、飘窗、北阳台、风口、柜体、设备带、机电管** 风险点
- 与 [A] 两联供、FP、ERV、卫间防水 **交叉对齐**

**PE 隔汽膜铺设、外墙剖面、插座/穿墙/飘窗节点不在此重复**，见 [02-墙体系统 §4、§10–§16](02-墙体系统.md)。

---

## 2. 适用范围

| 包含 | 不包含 |
|------|--------|
| 围护结构 **防结露**（隔汽 + 保温 + 冷桥） | 卫间 **液态防水** 工序（见 [03 §5–6](03-地面系统.md)） |
| 窗框、飘窗、北向高风险区 | 门窗产品选型（待 [05](05-门窗系统.md)） |
| 新风/空调/冷凝水管 **防结露保温** | 完整暖通水力计算（07/09 号） |
| 柜体背后、设备背后通风防潮 | 地下室（本项目无） |

## 3. 两类「潮湿」必须分开

| 类型 | 机理 | 本项目做法 | 文档 |
|------|------|------------|------|
| **气态结露** | 暖湿空气遇冷面 → 水珠 | 隔汽层 + 保温 + 断冷桥 | **本文件** + 02 §4–16 |
| **液态水** | 渗漏、毛细、积水 | 防水涂料/卷材、坡度、存水弯 | **03 卫间**、02 §8、11 冷凝水 |

**禁止** 用卫间防水涂料代替外墙隔汽层；**禁止** 只做防水不做隔汽。

## 4. 设计参数 · 安徽亳州（21 层）

> 依据 **DB34/1466** 类安徽省居住节能标准；亳州气象参考 **表 3.0.2「商丘/亳州」站**；暖通名义工况见 [09 §4.1](09-空气能及水系统.md)。

| 参数 | 数值 | 用途 |
|------|------|------|
| 冬季室内设计温度 | **18 ℃**（节能计算）；地暖舒适验算可用 **20 ℃** | 内表面温度验算 |
| 冬季采暖室外计算温度 | **-2.5 ℃** | 常规结露验算 |
| 冬季累年最低日平均温度 | **-7.7 ℃** | **保守验算** |
| 设计采暖期 | 约 **12/1–3/15**（2030 HDD18 ℃·d 量级） | 结露季 |
| 夏季室内设计温度 | **26 ℃** | 空调结露 / 新风 |
| 夏季空调室外计算 | **约 33 ℃**（夏热冬冷区） | 风管、冷媒管保温 |
| 室内相对湿度倾向 | 冬季采暖 **40–50% RH**；夏季空调 **50–60% RH** | 露点估算 |
| 露点控制原则 | 隔汽层应在 **保温层暖侧**；露点落在保温层内 | 02 §4.5 |

**简化结论（本项目）：**

- **南/北外墙、北封闭阳台原外立面、电梯墙** → 必须 **02 §4 全剖面**（隔汽连续）
- **东/西分户墙** → **只隔音不保温**，不做隔汽保温套系（02 §5）
- **21 层 + 计划外墙保温后**，内表面结露风险 **可控**，前提是 **隔汽无破口**

## 5. 标准构造 · 气态结露（索引）

```text
室内（暖侧）
  ↓
PE 隔汽膜（≥0.2 mm，搭接 ≥100 mm，胶带密封）
  ↓
减振垫 + 75 轻钢龙骨
  ↓
50 mm 憎水岩棉（≥60 kg/m³）
  ↓
结构墙（冷侧）
  ↓
室外
```

| 层次 | 不做会怎样 |
|------|------------|
| 隔汽层不连续 | 岩棉吸潮、保温失效、内侧发霉 |
| 无减振垫 | 声桥（兼 13 号） |
| 单层薄石膏板 | 内表面温度低 → 结露、防火不足 |

详 [02 §4.3–4.5](02-墙体系统.md)、[01 §4.4](01-总体设计原则.md)。

## 6. 风险分级矩阵

| 等级 | 位置 | 风险原因 | 应对（最低要求） |
|------|------|----------|------------------|
| **🔴 高** | **北封闭阳台原外立面** | 原室外变室内，温差大 | **02 §4 全做** + 地面防水找坡（02 §3.4.4、§26.1） |
| **🔴 高** | **飘窗** 顶/侧/窗框 | 多面暴露 | 02 §14；窗框发泡+密封；柜后通风 |
| **🔴 高** | **洗衣区** 湿区相邻 | 潮气、地漏 | 02 §26.2 实墙+防水门；墙脚防水上翻 ≥300 mm |
| **🟠 中高** | **北向房间外墙** | 日照少 | 02 §4；窗帘盒局部保温 |
| **🟠 中高** | **新风 FA 低位送风口** 周边 | 冷气流 | [08](08-新风系统.md) 保温；勿直吹窗框 |
| **🟠 中高** | **设备带** ac_hp/dhw/erv 背后 | 热桥、化霜水 | 09 §5；化霜水有组织排放（11 §10） |
| **🟡 中** | **南向窗带** R05 等 | 冬季温差小于北向 | 02 §4；窗框节点同 §14 |
| **🟡 中** | **柜体背后**（电视墙、厨柜、衣柜） | 通风差 | §8 柜体通则 |
| **🟡 中** | **FP 贴墙腔** | 冷凝水+冷管 | 07 冷凝水坡度；空调水管保温 |
| **🟢 低** | **东/西分户墙 T18** | 无室外温差 | **不保温**；防漏音穿孔密封即可 |
| **🟢 低** | **卫间内部** | 液态水为主 | **03 双防水+满铺地暖** |

## 7. 逐区域要点（结合当前设备）

### 7.1 北封闭阳台 · 洗衣区 · 衣帽间

| 项目 | 要求 |
|------|------|
| 原外立面 | **情形 A → §4 外墙全做** |
| 新建隔断 | §3.4.4 情形 B：100 轻质砖 + 50 岩棉 + 防潮双层板 |
| 地面 | 防水 + **≥1% 找坡** → 地漏（03、02 §26.1） |
| 洗衣机 `washer` | 地漏排水；墙脚防水；柜后 **20–30 mm 透气缝** |
| 衣帽间 | 背外墙 **不另砌**；柜体与墙 **防潮膜/透气缝** |

### 7.2 设备带（厨房后北墙）

> [`20260628_MEP设备位置.csv`](../融创大观-3D/模型/全屋/方案推敲/20260628_MEP设备位置.csv)

| 设备 | 防潮/结露要点 |
|------|---------------|
| [A] `ac_hp` | 化霜水 **有组织排放**；底座减振；检修面净空 |
| [B] `dhw` | 与 [A] 分离；机旁无积水；冷凝/滴水接排水 |
| `erv` | 室内段风管 **保温 ≥20 mm**（08）；主机旁不积水 |
| `fh_manifold` | 柜内结露风险低；管道保温见 09 §6.2 |
| 穿墙管 | 套管 + 发泡 + 密封 + **隔汽补强** |

### 7.3 五台 FP + 空调水 + 冷凝水

| 项目 | 防结露 |
|------|--------|
| 二次侧 PPR | 保温橡塑 **≥20 mm**（09 §6.2） |
| FP 支管 DN25 | 同左 |
| 冷凝水 PVC | 坡度 ≥1%；存水弯；穿越非空调区可选 **9 mm 保温**（11 §4） |
| FP 贴墙 | 背板留 **通风**；接水盘无积水 |

### 7.4 新风 FA1–FA5

- 风管：**北向段、穿越非空调区** 橡塑 **≥20 mm**（08 §6）
- 外墙进排风孔：套管 + 密封 + **隔汽补强**（02 §13、08）
- 低位送风口：避开窗框正下方冷点

### 7.5 卫间（液态水 + 地暖）

- **工序**：找坡 → 一防水 → 保护 → 地暖 → 带压回填 → 二找坡 → **二防水** → 铺砖（03 §5）
- **满铺地暖**；门槛 **±0** 齐平干区；止水坎
- 与 **气态结露** 分工：卫间靠 **防水层**；外墙靠 **隔汽层**

### 7.6 厨房 · 水柜

- `cabinet` 厨下：RO 浓水、软水再生 → DRAIN-HUB（10 §5.1）
- 厨柜背后：**10–20 mm 透气缝** 或防潮膜
- 开放厨房：蒸汽、油烟 — 油烟机 **独立排**；FA5 送风避灶台（08）

## 8. 柜体背后 · 通风通则

| 场景 | 做法 |
|------|------|
| 电视墙网络柜/矮柜 | 背板 **不完全封死**；**≥20 mm** 透气缝或格栅 |
| 厨房吊柜/地柜 | 背板 **防潮膜**；距墙 **10–20 mm** |
| 衣柜/衣帽间 | 贴外墙侧：**透气缝 + 防潮膜**；禁 **0 间隙** 吸墙 |
| 飘窗柜 | **预留通风** 或局部保温（02 §14.2） |
| FP 落地背板 | 贴 OSB 但腔体 **不积水**；接水盘常检 |

## 12. 施工顺序（与 02 对齐）

```text
1. 结构墙界面剂
2. PE 隔汽膜整铺 → 搭接/胶带/穿孔补强 → 验收拍照
3. 减振垫 + 龙骨 + 岩棉
4. 穿墙套管 + 机电管预埋（同步保温套）
5. 局部 OSB → 双层石膏板
6. 卫间/洗衣区：液态防水（03）与外墙隔汽 **分工施工**
7. 设备就位 → 化霜/冷凝水通水试验
8. 封板前：隔汽连续 + 冷桥节点 **终检拍照**
```

---

## 13. 禁止事项

- 外墙 **无隔汽层** 或 **破口未补** 即封板
- 用 **XPS 全屋内保温** 替代岩棉+隔汽（02 §19）
- 分户墙 **误做外墙保温**（浪费且可能结露界面错）
- **背靠背插座**（外墙、分户墙）
- 穿墙 **无套管**、破坏隔汽 **不恢复**
- 柜体 **0 间隙吸外墙**、飘窗柜 **无通风**
- 冷凝水、化霜水 **无坡度** 或 **直滴平台**
- 新风/空调管 **裸管** 穿越非空调区或北向段
- 卫间 **只做一道防水** 或 **跳过闭水**（03）

## 14. 隐蔽验收

- [ ] 隔汽膜连续、搭接 ≥100 mm、穿孔已补强（**强制拍照**）
- [ ] 北阳台外立面 / 洗衣区：**§4 外墙** 或 §3.4.4 情形 B
- [ ] 冷桥节点：窗框、套管、插座（无背靠背）
- [ ] 空调水/新风管保温厚度
- [ ] 冷凝水坡度 + 存水弯（11）
- [ ] 卫间双防水闭水记录（03）
- [ ] 柜体背后透气缝方案交底
- [ ] 设备带化霜/冷凝排水试通
