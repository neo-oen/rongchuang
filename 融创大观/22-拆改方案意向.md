## 3. 当前拆改方案（设计中1 · 方案 E）

> **来源：** `Downloads/融创大观/04_方案推敲/设计中1.skp` → 项目模型 `20260627_设计中1_v01.skp`

> **说明：** 以下为相对 **§1 原始户型** 的定向拆改，已非 A/B/C/D 四选一探讨。

### 3.1 拆改后顶视（当前模型）

![拆改后顶视](../融创大观-3D/模型/全屋/方案推敲/20260627_设计中1_顶视分区参考.png)

**读图要点：**

- **黑色** = 承重墙（禁止擅自拆）
- **灰 / 白** = 普通填充墙（拆改候选，须结构图确认）
- **浅蓝** = 玻璃 / 窗带
- **西外轮廓 T18**、**东外轮廓 T8 / R34 系列** = **分户墙**（左右邻居接触面）
- **南/北外轮廓 + 北阳台外缘** = **外墙**

### 3.2 整体策略（一句话）

**北侧阳台整体进室 + 左入户家政模块（洗衣区/衣帽间）+ 厨房开放 + 次卧北扩 + 书房扩大 + 主卫改门向**；主卧区基本保持。

### 3.3 分区拆改一览

| 区域 | 相对原始户型 | 拆改动作 | 拆改后用途 |
|------|--------------|----------|------------|
| **北 · 阳台** | 室外阳台 | **整体封闭**进室内 | 室内地面：防水 + 找坡 + 保温；原外立面按外墙标准 |
| **西北 · 洗衣区** | 原阳台西段 | 纳入室内 | 洗衣机 + 高柜（R39A/R43/R48 夹角） |
| **西 · 衣帽间** | 原无独立区 | **新建围合**（左右柜体，中间不设墙） | 入户竖向衣帽间；背面贴外墙，不另砌堵墙 |
| **西南 · 玄关+厨房** | L 形隔墙分隔 | **拆除面向公区的普通隔墙** | 开放厨房 + 玄关视觉贯通 |
| **厨房 ↔ 公区** | R19 位置隔墙 | **拆除 R19、R20** | 改为双面柜 + 开放台面（v05 意向） |
| **客厅** | 不变轮廓 | 北缘接封闭阳台 | 公共活动区扩大进深 |
| **北 · 次卧** | 原较深较窄 | **拆 R35A/R31 等**，向北吃进阳台 | 次卧面积增大 |
| **次卧南界** | 原矮墙/门槛 | **新建带门洞隔墙**（R35B→R37 段） | 次卧与下方空间重新分隔 |
| **次卧东界** | 原 R42A/R49 | **拆除** | 与书房/通道关系调整 |
| **书房** | 原较小 | **新建白墙扩大**（R38→R54 意向） | 独立书房加大 |
| **主卫** | 原门向 | **门洞改向**（面向 suite 走廊） | 主卧套房动线 |
| **主卧** | — | **基本保留** | 不设内部衣帽间 |

### 3.8 拆后新建墙怎么弄（四类体系）

> 详细剖面、材料、厚度、禁止项见 **[02-墙体系统 §3.4](../02-墙体系统.md#34-新建墙四类施工体系拆改后--2026-06-27)**。

| 四类 | 本项目对应位置 | 一句话做法 |
|------|----------------|------------|
| **① 南向窗带边界** | 客厅/次卧/主卧 **南向落地窗侧边**（R05 等） | 100mm 轻质砖/ALC + 窗边欧松板；噪音大再加岩棉双层石膏板 |
| **② 室内普通墙** | 书房新墙、次卧门洞新墙、房间之间 | 75 龙骨 + 50 岩棉 + 双面双层 12mm 石膏板 |
| **③ 北向封闭阳台** | 北阳台、洗衣区、客厅/次卧靠北段 | **原外立面**按 §4 外墙全做；**新建隔断**半外墙加强 + 墙脚防水 |
| **④ 衣帽间薄墙** | 衣帽间面向玄关/走廊的新建堵墙 | 50 龙骨 90–100mm + 18mm 欧松板；背面结构墙不另砌 |

**特别说明：**

- **东/西分户墙**（T18、T8/R34 系列）→ 只做 [§5 分户墙隔音](../02-墙体系统.md#5-分户墙系统)，**不做外墙保温**
- **南/北原黑色结构外轮廓 + 北阳台外缘** → 直接在结构墙上做 [§4 外墙](../02-墙体系统.md#4-外墙系统)
- **洗衣区 ↔ 衣帽间** 水平隔断 → [§26.2.2 厨卫墙 + 防水门](../02-墙体系统.md#2622-洗衣区--衣帽间水平隔断厨卫墙--门)，不是简单薄石膏板

**全局禁止：** 单层 9.5mm 石膏板、空腔不填岩棉、穿墙孔留空、封板后破坏防潮层。
